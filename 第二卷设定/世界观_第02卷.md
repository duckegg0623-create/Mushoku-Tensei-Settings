# 世界观设定（第02卷）

## 世界历史年表

### 太古的神之时代（十万年以上前）
- 世界分为七个世界，各自受不同的神明支配：
  - 人的世界 → 人神
  - 魔的世界 → 魔神
  - 龙的世界 → 龙神
  - 兽的世界 → 兽神
  - 海的世界 → 海神
  - 天的世界 → 天神
  - 无的世界 → 无神
- 各世界被类似结界的东西隔绝，无法自由往来
- 只有一部分神明和力量强大到能突破结界者才知道其他世界的存在

### 两万～一万年前
- 龙的世界诞生出邪恶的龙神，打破结界
- 率领「五龙将」毁灭其他世界
- 只剩最后一个世界时，五龙将背叛龙神
- 龙帝与四名龙王与龙神死斗，不分胜负
- 龙的世界崩毁，最后只剩下人的世界（即现在的世界）

### 混沌时代（一万～八千年前）
- 人族祖先与其他世界居民互相交战
- 各族分别栖身：兽族住森林、海族支配大海、天族掌控高地、龙族隐居、无族无处不在
- 人族与魔族在平地争斗
- 中央大陆与魔大陆相连，称为「巨大陆」

### 约七千年前
- 第一次人魔大战爆发，人族与魔族大规模冲突
- 牵连其他种族的漫长战争

### 约六千年前
- 勇者亚尔斯率领六名同伴打倒「五大魔王」和「魔界大帝奇希莉卡」
- 持续千年的人魔大战结束

### 约五千五百年前
- 人族误以为自己强大，与其他种族交战，人族内部也内战
- 魔族被当成奴隶对待
- 战国时代持续约五百年

### 约五千年前
- 第二次人魔大战爆发
- 魔界大帝奇希莉卡率领魔族反击（奇希莉卡为不死身，死后数百年复活）
- 魔族拉拢兽族和海族，人族被逼到绝境

### 约四千两百年前
- 黄金骑士阿尔德巴朗一人击退超过一万敌军
- 施展大招将巨大陆打出大洞，分为中央大陆和魔大陆，创造出林古斯海
- 有一种说法认为阿尔德巴朗就是人神
- 第二次人魔大战终结，和平到来

### 约四千两百年～一千年前
- 人族用外交手段将魔族关进魔大陆（花了三千年以上）
- 魔族在严苛环境中内乱，人数减少但锻炼得更强

### 一千年前
- 魔神拉普拉斯诞生（历史上唯一被称为魔神的存在）
- 统一魔大陆，建立类似魔界统一帝国的组织
- 至今仍是魔大陆的偶像

### 五百年前
- 拉普拉斯战役爆发
- 拉普拉斯说服海族和兽族后攻入中央大陆
- 在中央大陆放出赤龙阻断山区，从南北两面侵略人族

### 四百年前
- 七名英雄说服海族解除大海封锁，沿海路前往米里斯大陆
- 说服兽族（实际手段包括绑架小孩和恐吓）
- 决战当日，七英雄率领米里斯圣骑士团和兽族奇袭拉普拉斯大本营
- 四名英雄战死，成功封印魔神拉普拉斯
- 存活三人：「龙神乌尔佩」、「北神卡尔曼」、「甲龙王佩尔基乌斯」
- 被称为「杀死魔神的三英雄」（实际上没有杀死，是封印）
- 人族与稳健派魔王缔结条约，解除魔大陆封锁，禁止歧视魔族

### 现代
- 中央大陆歧视魔族风气依然存在，但基本和平
- 数字「七」为吉利数字（受七英雄、七世界影响），「六」为不吉利数字

## 种族
- 长耳族(Elf)、矿坑族(Dwarf)、小人族(Hobbit)等各种种族
- 分类上被视为魔族，也有一说认为是在混沌时代诞生的新种族
- 有些种族没有寿命限制（如魔界大帝、不死身魔王）
- 可能也存在能让身体变成不死身的魔术

## 世界语言

| 语言 | 使用区域 |
|------|----------|
| 人类语 | 中央大陆 |
| 兽神语 | 米里斯大陆北部 |
| 斗神语 | 贝卡利特大陆 |
| 天神语 | 天大陆 |
| 魔神语 | 魔大陆 |
| 海神语 | 所有海域 |

- 语言名称基于各大陆信仰的神
- 中央大陆分北、西、南三部，人类语略有方言差异（类似美式/英式英语差别）
- 西部的人类语使用者被视为有钱人，在其他地区需注意安全
- 大海中居住着「海人族」

## 剑术流派等级体系

- 剑神流等级（从低到高）：初级 → 中级 → 上级 → 剑圣 → 剑王 → 剑帝 → 剑神
- 剑帝有两人，剑王只有一人（基列奴是剑神流第四强）
- 剑神流奥义：「光之太刀」（剑速达光速）
- 另有北神流、水神流两大流派
- 「斗气」是成为上级剑士的关键

## 阿斯拉王国

### 四大地方领主家族（格雷拉特家分支）
- **诺托斯(Notos)** — 保罗出身家族
- **伯雷亚斯(Boreas)** — 菲托亚领地领主，艾莉丝的家族
- **艾乌洛斯(Euros)** — 另一分支
- **泽费洛斯(Zephyros)** — 另一分支

### 伯雷亚斯家传统
- 所有出生男孩由下任当家养育（夺走败者之子）
- 目的：防止权力斗争中的败者拥立儿子参与下一代争斗
- 绍罗斯有十个儿子，其中菲利普、高登、詹姆士三人竞争下任当家
- 詹姆士获胜（靠人脉、金钱、权力，以及六年年龄优势）
- 菲利普被安排为罗亚市长，高登入赘艾乌洛斯家
- 现任诺托斯当家皮列蒙·诺托斯·格雷拉特为人糟糕，有人试图换他下台

## 菲托亚领地 / 罗亚要塞都市
- 罗亚：菲托亚领地的要塞都市
- 伯雷亚斯家宅邸位于此
- 领地内有布耶纳村（保罗一家住处）
- 格雷拉特家宅邸内设有书库，收藏历史书等书籍

## 魔力灾害（菲托亚领地消失事件）
- 发生在甲龙历四一七年（鲁迪乌斯十岁时）
- 天空出现异色光芒（紫色、褐色、黑色、黄色的大理石纹路）
- 大规模的魔力灾害——实际上是转移事件
- 菲托亚领地所有建筑物和树木消失，居民被传送至世界各地
- 部分人被传送到山上、半空中、海里而立即死亡
- 被称为「菲托亚领地消灭」的大灾害
- 引起了世界各地强者的注意（龙神奥斯帝德、甲龙王佩尔基乌斯、剑神加尔·法利昂、魔界大帝奇希莉卡等）

## 其他设定
- **马尔齐渊佣兵国**：中央大陆纷争地带的小国，靠派遣佣兵为生
- **森之女神列奴**：马尔齐渊的守护神，原型为基列奴（发音讹变）
- **魔石与魔杖**：魔石越透明越大效果越好，有颜色的魔石能增幅对应系统。魔力结晶与魔石不同，用于魔道具和大型魔术
- **厄禁森林**：马尔齐渊附近有大量魔物出没的危险森林
- **赤龙山脉**：中央大陆的山脉，赤龙栖息地
- **魔界大帝奇希莉卡**：别名「魔眼的魔帝」，拥有十种以上魔眼，不死身，目前复活但无人理睬
