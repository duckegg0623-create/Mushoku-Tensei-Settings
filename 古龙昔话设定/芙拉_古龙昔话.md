# 芙拉·克罗蒂亚/Fula Crotia（古龙昔话·王龙王讨伐篇）

## 基本资料
- 名称：芙拉·克罗蒂亚（Fula Crotia）
- 称号：流云——魔法行会「草世叶木」所属
- 身分：战斗魔法师（中级）——后成为魔法大学初代校长——第五十六代魔术公会总帅
- 外貌：白里透蓝的头发——面无表情——常被误以为在发呆——实则深思熟虑
- 本卷状态：经历王龙王讨伐后——返回行会——洗心革面重新修行——最终实现梦想

## 魔法才能
- 3岁被判定有魔法才能——拜入最高级魔法师门下
- 6岁正式学习魔法——10岁初级魔法师——13岁中级魔法师
- 16岁已获得上级魔法师的技术认可——只差正式考试
- 草式战斗魔法：向手足装填魔法「子弹」——半咏唱——随时释放
- 擅长魔法：
  - 火散弹——水墙——水波斩——移动传送——连射火弹——祈雨
  - 蒸发魔法（零距离一击必杀——但无法近身王龙王）
- 致命弱点：体力不足——师傅一再说「先增强体质」

## 黑暗的一面
- 魔法行会「草世叶木」分日派与月派——她的师傅是月派高级干部
- 月派被日派逼到墙角——芙拉急于通过上级考试
- 暗中支付巨额酬金给马努库鲁——约好铲除日派首领「日出」和日派干部
- 马努库鲁超额完成任务——将整个魔法团全灭——芙拉自己也险些暴尸荒野
- 被亚尔救下——体力魔力耗尽躲于树洞中
- 加入亚尔队伍——既是为了报恩——也是因为空手回去会被怀疑

## 王龙王讨伐中的角色
### 魔法师的价值
- 两周间用探索魔法确认王龙王的方位
- 在王龙巢穴的战斗中——一开始完全无法发挥作用——体力不足
- 关键时刻：
  - 用水墙抵消龙息救了缇珂
  - 用移动传送扑救夏伊娜——把她从龙爪下拉出
  - 崩塌洞顶掩护大家逃跑——但被王龙王的重力魔法化解
- 深刻认识到自己的不成熟——回想起师傅的忠告

### 隐藏与暴露
- 在逃亡时与马努库鲁相对——被马努库鲁用她的秘密威胁
- 马努库鲁：「我还有副官幸存——我死了——他会去魔法行会报信」
- 马努库鲁掌握了芙拉与日派被灭的关系——握住她的把柄
- 夏伊娜因此饶了马努库鲁——「云酱可能会成为我们的敌人」
- 马努库鲁承诺对此保密——两人暂时休战

## 决斗后
- 在工坊停留三天——照顾夏伊娜（两天不眠不休）
- 返回行会前——在城门外看到入定冥想的亚尔——遥遥鞠躬致礼
- 决心洗心革面：不为头衔——为成为关键时刻能靠得住的人
- 首先从增强体力开始
- 通过青豹传话给亚尔：「如有需要魔法师的时候尽管来找我——随叫随到」

## 后续
- 成为魔法大学初代校长——在魔法都市夏利亚的中心广场立有她的雕像
- 「初代校长——第五十六代魔术公会总帅芙拉·克罗蒂亚」
- 在正传中——鲁迪乌斯在魔法大学看到的就是这尊雕像
- 她的梦想是在全世界建立孤儿院——这与她建立魔法大学的动机密切相关

## 备注
- 芙拉是古龙昔话中唯一一个在正传世界里留下明确痕迹的人物——鲁迪乌斯每天都能看到她的雕像
- 她从自私的野心家到真正成熟魔法师的成长——是王龙王讨伐篇写得最好的暗线
- 她并非单纯的"好人"或"坏人"——她为了自己的目的不惜买凶杀人——但也为了刚认识两周的同伴拼上性命
- 她的体力不足的问题来自师傅的反复提醒——这成了整个故事的伏笔——最终她正视了自己的弱点
