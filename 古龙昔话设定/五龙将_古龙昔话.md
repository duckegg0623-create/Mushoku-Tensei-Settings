# 五龙将（古龙昔话）

## 基本资料
- 名称：五龙将（Five Dragon Generals）
- 身分：龙神直属的最高干部——各自统领一个龙族氏族
- 制度起源：龙神在授予龙族龙气使用方法后——选出五龙将——分掌龙界军政
- 本卷状态：全部死亡——仅拉普拉斯活到正传时代（但已分裂失忆）

## 历代五龙将构成

### 初代（露娜莉亚死前）
- 圣龙帝席拉德（总统领、龙士指挥、龙神代理）
- 冥龙王马克士威（魔物退治、冥龙队指挥）
- 狂龙王卡奥斯（武器制造、龙神刀等神具锻造）
- 刚龙王克里斯塔（召唤/转移研究）
- 甲龙王朵拉（龙的调教）

### 改组后（克里斯塔死后）
- 圣龙帝席拉德（不变）
- 冥龙王马克士威（不变）
- 狂龙王卡奥斯（不变）
- 甲龙王朵拉（继承召唤/转移研究——同时负责龙的调教）
- 魔龙王拉普拉斯（末席——龙神辅佐/外交——接替朵拉原职）

## 各成员详述

### 圣龙帝席拉德（Shirad）
- 五龙将之首——龙神不在时的代理
- 对龙神绝对忠诚——"席拉德的忠诚"是Hitogami利用的核心
- 在众神会议上率先提案搜查杀害魔帝的凶手
- 被Hitogami操纵——推动五龙将反抗龙神：「一切都是为了龙神大人」
- 五龙将vs龙神战后——Hitogami露出真面目——当场自杀谢罪
- 龙神临终说：「席拉德是忠臣。不许笑」
- 讽刺：他越是忠诚——越是被Hitogami精确地利用

### 冥龙王马克士威（Maxwell）
- 负责魔物退治——冥龙队指挥
- 派遣部下柯拉与斯克拉维协助拉普拉斯调查克里斯塔之死
- 参与五龙将反抗龙神——战败后为拖延龙界崩坏而死
- 龙神原谅了他：「都是因为我没有跟你们说清楚」
- 他的龙门炮（神具）参与了毁灭龙鸣山的战斗

### 狂龙王卡奥斯（Chaos）
- 武器制造者——锻造龙神刀等神具
- 龙界首都"凯欧斯"（Chaos）以他命名——龙鸣山凯欧斯
- 对拉普拉斯成为五龙将最激烈的反对者——体现了龙族对混血儿的偏见
- 但在拉普拉斯证明自己后——态度缓和——"连卡奥斯也收敛了不少"
- 参与五龙将反抗龙神——战败后为拖延龙界崩坏而死
- 龙神同样原谅了他

### 刚龙王克里斯塔（Crista）
- 第一位被杀害的五龙将——死在露娜莉亚之前
- 负责召唤/转移研究——龙界机密项目的核心
- 被涅克洛利亚纳克洛利亚（八大魔王之一）杀害——脑髓被吸走
- 尸身在金龙巢附近被发现——看似被金龙所杀——实际是魔王奉魔神之命所为
- 他的死是Hitogami阴谋的第一环——目的是阻止龙界的转移研究成果公开
- 死后——拉普拉斯被席拉德提名负责追查

## 五龙将的悲剧
- 全部五龙将都是忠诚的——没有一人是背叛者
- Hitogami的手法：利用他们的忠诚——让他们自己走向违背龙神意愿的方向
- 五龙将vs龙神的战斗——是忠诚与忠诚的冲突——是悲剧中的悲剧
- 龙神知道真相后没有责骂他们——反而自责：「都是我的不德」
- 仅拉普拉斯一人活到正传时代——但分裂为魔神与技神——失去了一切记忆

## 备注
- 五龙将制度是龙神"全员都是我的孩子"信念的制度化体现——龙神对五龙将的信任是绝对的
- Hitogami正是利用这一点——让五龙将相信阻止龙神"是为了龙神好"
- 龙鸣山凯欧斯——龙界首都——最终在五龙将与龙神的战斗中被完全摧毁——象征了龙族文明的终结
- 五龙将的后裔（佩尔基乌斯、奥尔斯帝德、以及持有秘宝的各氏族）延续到了正传时代
