# 希露菲·格雷拉特（六面世界3篇）

## 基本资料
- 名称：希露菲·格雷拉特（白妈妈）
- 身分：鲁迪乌斯的第一位妻子——齐格与露西的生母
- 本卷状态：作为齐格的生母——最温柔地目送儿子离家——忍不住落泪

## 对齐格的关怀
- 齐格无职宅家的一年里——每天叫他起床——不是休息日一定准备早餐
- 齐格做正义的伙伴晚归时——偶尔会起床迎接
- 最先注意到齐格在躲父亲——在早饭桌上看着空座位叹气
- 也是最先察觉儿子不对劲——但遵循「暂时放着比较好」的方针

## 齐格的告别
- 齐格宣布要远行后——白妈妈咬住嘴唇——马上要哭出来
- 大姐露西已远嫁米里斯——齐格也要走的话——两个亲生孩子都离家了
- 听齐格说「至今受到大家照顾了」——终于忍不住泪珠巴拉巴拉掉下来
- 但哭着说：「我很高兴啊——齐格能好好的走向下一步了」
- 红妈妈和蓝妈妈安抚她：「要笑着目送他啊」

## 对齐格的影响
- 小时候苦口婆心教育「结交朋友——不要欺负弱小」
- 齐格对正义的伙伴的憧憬——来自父亲讲的芝士超人故事
- 但觉得这件事「很帅」——是白妈妈教导的结果

## 备注
- 希露菲在本卷展现了平凡母亲最动人的一面——为孩子的成长真心喜悦而忍不住流泪
- 与女儿露西和儿子齐格先后离家——暗示了格雷拉特家逐渐空巢的过程
