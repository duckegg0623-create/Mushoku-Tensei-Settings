# 艾莉丝·格雷拉特（六面世界3篇）

## 基本资料
- 名称：艾莉丝·格雷拉特（红妈妈）
- 身分：鲁迪乌斯的第三位妻子——亚尔斯与克莉丝汀娜的生母——剑王
- 本卷状态：在齐格告别时展露笑容——肯定儿子的决心

## 对齐格的教育
- 一直教导兄弟姐妹们「保护某个人是很重要的」
- 齐格作为她丈夫和希露菲的孩子——依然被平等对待
- 对齐格无职期间——和平常三位母亲中最常皱眉——因为看不惯孩子停滞不前

## 齐格的告别
- 当齐格宣布要去帮助朋友——红妈妈最先开口：「那个友人是你想保护的人吧？」
- 听到确认后——「下定决心了吧——加油」
- 露出久违的笑容——齐格记忆中最近总是看她皱眉
- 在他停滞不前时——红妈妈最不能容忍——但当儿子真正「保护别人」的决定出现——她毫无保留地支持

## 与其他母亲的对比
- 白妈妈：哭泣——感情最丰富——代表不舍
- 蓝妈妈：理性——代表尊重选择
- 红妈妈：坚定——代表对「战斗/保护」这一信念的认可

## 备注
- 艾莉丝虽非齐格生母——但她的教导贯穿了齐格的成长轨迹
- 在齐格决定「保护帕库斯」的那一刻——他终于符合了红妈妈一直以来的期待
- 三位母亲共同送别的场景——是格雷拉特家独特的家族情感缩影
