# 菈菈·格雷拉特（六面世界3篇）

## 基本资料
- 名称：菈菈·格雷拉特
- 身分：鲁迪乌斯与洛琪希的女儿——格雷拉特家第二子（二姐）——魔术大学研究者——占卜/预言术专攻
- 本卷状态：在研究室研究不明魔道具——送别齐格——罕见展现笑容与认可

## 性格
- 平常总是想睡觉的无表情——看起来对什么事都不关心
- 说话冷淡——对齐格常叫「笨蛋」——不参与家庭情感表达
- 实际是兄弟姐妹中最敏锐的——用占卜知道齐格迟早会走上这条路
- 比外表看起来更关心家人——对齐格的「正义的伙伴」知情但不干涉

## 与齐格的关系
- 齐格无职期间——常到菈菈的研究室消磨时间
- 菈菈从不赶他走——让他自由使用房间
- 从不明说但一直在观察弟弟的状态

## 不明魔道具的研究
- 在市场找到用途不明的魔道具——外观圆形有棱角——表面刻有魔法阵
- 妹妹莉莉对此极度着迷——从昨天起窝在研究室一起研究
- 精疲力尽地应付莉莉的热情——一边继续研究

## 与齐格的告别
- 当齐格来道别——说「我早觉得有一天会变成这样」
- 对齐格的「正义的伙伴」直接点破：「不继续做那个了吗？」
- 问：「打算不再帮助别人——而要变成坏人吗？」
- 齐格回答「堂堂正正的——帮助我想帮助的人」后——罕见地露出笑容
- 「今天的齐格很聪明」——第一次没有叫笨蛋
- 最后叮嘱：「即使你去了远方——即使过去很多年——我们大家一直是兄弟姐妹——这点不要忘记了」

## 备注
- 菈菈的占卜能力暗示了她知道更多命运的走向——但选择不说
- 「即使过去很多年」暗示了长寿种族——作为洛琪希（魔族）的女儿——她有漫长的寿命
- 对齐格的认可是本卷最克制的温情瞬间——对比三位母亲的情绪化告别
