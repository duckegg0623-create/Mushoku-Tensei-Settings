# 洛琪希·格雷拉特（六面世界3篇）

## 基本资料
- 名称：洛琪希·格雷拉特（蓝妈妈）
- 身分：鲁迪乌斯的第二位妻子——菈菈与莉莉的生母——魔族
- 本卷状态：为齐格张罗与莎莉爱尔相亲——最终接受儿子的决定

## 为齐格的相亲
- 因与爱丽儿女王关系好——主动推动齐格与莎莉爱尔的婚约
- 希望通过联姻强化格雷拉特家与阿斯拉王国的关系
- 但当齐格明确拒绝——洛琪希大度接受：「我一直说你想怎么活就怎么活——既然是你决定的道路——就走下去就好了」
- 对齐格的告别露出笑容——展现了作为母亲的开明

## 教育方针
- 教给孩子们「随心所欲的活着——有不懂的东西就问问别人」
- 齐格虽遵守了这条教诲——却无法为自己的人生做出选择——直到面对父亲

## 其他出场的含蓄描写
- 在这之前为丈夫鲁迪乌斯收集了20多人的婚约肖像——其中一个成功组成了婚姻
- 在婚礼上帮忙——看到哭泣的新娘——展现出对人际关系的理解

## 备注
- 洛琪希的蓝妈妈角色虽出场较少——但在齐格拒绝婚约时的反应展现了她教育理念的一贯性
- 作为魔族长寿种族——她与希露菲不同——对孩子的远行有更长的时间视角
- 菈菈对弟弟说的「即使过去很多年」——也继承自母亲的这份时间观
