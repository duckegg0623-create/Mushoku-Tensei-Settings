# 莉莉·格雷拉特（六面世界3篇）

## 基本资料
- 名称：莉莉·格雷拉特
- 身分：鲁迪乌斯与洛琪希的女儿——格雷拉特家第五子——魔道具开发者——札诺巴的助手
- 本卷状态：沉迷于调查不明魔道具——得知哥哥要走后愤怒——最后悻悻接受

## 性格与日常
- 对魔道具有异常热情——见到用途不明的魔道具会极度兴奋
- 缺乏方向感——常迟到——但靠着娘家背景或成果保住了工作
- 天然呆——容易被坏人盯上——小时候齐格常保护她
- 嘴硬心软——用愤怒掩饰寂寞

## 对哥哥的依赖
- 一直盘算着让齐格当自己的助手——「齐格哥哥力气那么大——还会骑马——也会娇惯我——是最适合的助手」
- 甚至已跟主任说了「明年新进一人」——完全单方面决定
- 经常让齐格骑马送迟到她去上班
- 齐格在阿斯拉上学时——她写信说要哥哥来当助手

## 不明魔道具
- 菈菈在市场找到——用途不明的球状物——表面刻有魔法阵——有棱角
- 莉莉为此兴奋不已——在研究室窝了一夜
- 「就是！看不出来是干什么的呢！这个东西真厉害！」
- 虽翘了工作但自称「这也是业务的一环」

## 告别反应
- 听到齐格要走——从惊讶变成愤怒
- 「那做我的助手的事怎么办？」——「我都跟主任说了明年新进一人——可怎么办啊？」
- 被齐格说「我从来没答应过」——噘嘴道：「切~」
- 小声说：「我还以为你绝对最后会因为缺钱而来当我的助手呢」
- 最后才流露真实情绪：「哥哥不在的话会寂寞起来啊——再也没法迟到了」
- 齐格提醒「就是我在你也别迟到」——「我又不是你的马」

## 备注
- 莉莉是兄弟姐妹中情感表达最笨拙的一个——用愤怒掩饰依恋
- 齐格走后——齐格介绍乔治去莉莉手下做助手——延续了情报屋与格雷拉特家的联系
- 莉莉的母亲是洛琪希——她也继承了一半的魔族血统——拥有长寿
