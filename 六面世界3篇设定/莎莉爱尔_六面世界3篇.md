# 莎莉爱尔（六面世界3篇）

## 基本资料
- 名称：莎莉爱尔
- 身分：阿斯拉王国公主——爱丽儿女王之女——齐格的婚约候选
- 本卷状态：向齐格提出婚约暗示——被回避——最终维持朋友关系

## 与齐格的关系
- 在阿斯拉王立学校相识——对齐格抱有好感
- 爱丽儿女王与鲁迪乌斯都想强化阿斯拉王家与格雷拉特家的关系——推动两人的婚约
- 洛琪希（蓝妈妈）充当了沟通桥梁——安排了相亲性质的接触
- 莎莉爱尔主动向齐格表示好感——但齐格从未回应

## 齐格的拒绝
- 齐格对父亲坦白：「并不讨厌——是个好朋友」
- 但两人无法真正沟通：「两个人独处的话——对话持续不长——沉默又很痛苦」
- 「结婚的话痛苦只会持续下去」
- 齐格的拒绝并非针对莎莉爱尔本人——而是他无法勉强自己走入不想要的人生
- 对齐格而言——保持朋友关系是最好的选择

## 备注
- 莎莉爱尔在齐格离开魔法都市夏利亚后未再出场
- 这段未成的婚约反映了子女一代的政治联姻压力——但鲁迪乌斯选择了尊重儿子的意愿
- 莎莉爱尔是爱丽儿的孩子——延续了阿斯拉王室线的世界观
