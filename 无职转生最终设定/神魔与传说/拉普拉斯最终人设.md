# 拉普拉斯（Laplace）

## 基本信息
- **全名**：拉普拉斯（Laplace）——后被授予「魔龙王」称号
- **称号**：魔龙王（五龙将末席） → 魔龙神拉普拉斯 → 分裂为「技神拉普拉斯」与「魔神拉普拉斯」
- **种族**：龙族与魔族混血
- **性别**：男
- **外貌**：绿色与银色交杂的斑纹头发（会动，令人忌讳）；金色眼睛；有翅膀、牙、爪；肤色白得像透明；鳞片极少
- **寿命**：活了几万年以上
- **所属**：龙界五龙将末席 → 最后的五龙将 → 在人界为奥尔斯帝德铺设道路

## 外貌特征演变
- **古龙昔话·野生期**：人形魔兽——被魔族视为怪物
- **古龙昔话·龙界新生期**：被龙神用血治愈，学会龙族生活方式
- **古龙昔话·五龙将期**：翅膀牙爪的混血姿态，作为五龙将末席活跃
- **古龙昔话·最终期**：失去全部记忆，再也记不起罗斯蒂娜，再也没有回到龙鸣山的家
- **第25卷**：已分裂为技神与魔神——间接通过巴迪和瑞杰路德叙述

## 性格与心境演变

### 野生期——孤独的深渊
- 诞生于魔界一角——不知父母、不知何时出生——有记忆起便独自生活在洞窟
- 靠捕食魔兽维生——天生强大力量——能单杀小山一样大的魔兽
- **关键转折**：第一次见到魔兽的家庭——被孤独感压垮——飞奔寻找同类
- 发现魔族城镇——却被视为「人形魔兽」——遭魔王涅克罗斯拉克罗斯暴力驱逐
- 数百年间不断靠近城镇、被赶走、再靠近——「比起痛楚，孤独更难忍受」
- 被未知超强魔物袭击——受致命伤——濒死之际被龙神所救

### 龙界新生期——找到归属
- 龙神用血治愈带回龙界——命名为「拉普拉斯」
- 露娜莉亚说：「作为龙神大人的养子，你也算是我的孩子了」
- 初期被佣人们当作野兽警戒——但自认龙神家的末席——每天巡逻守护大屋和露娜莉亚
- 露娜莉亚视他如己出——教他用餐具、准备适合无鳞肌肤的刷子、定制衣服
- 将露娜莉亚当作自己的母亲
- 接受龙神命令——朵拉（甲龙王）担任教官

### 成长与修行期
- 最初对朵拉威吓——被佣人们围住——感受到再次孤独——露娜莉亚保护了他
- 理解上下关系——做出最敬礼——朵拉接受了他
- 一年精通龙神语（通常需十年）
- 数年内学会龙族全部一般常识：文字、历史、世界、立场、礼仪、产业
- 战斗训练：每天被打到昏迷——飞行训练发展出独特的垂直飞行方式
- 数十年完成训练——朵拉评价「你已经是能独当一面的龙族了」

### 龙之调教师——获得立足之地
- 成为朵拉下属——负责调教赤龙
- 第一只龙萨列亚托：调教失败的个体——极度恐惧龙族
- 拉普拉斯的方法：不起名字不用暴力——起名「萨列亚托」——一起绝食——用牙咬碎肉喂它
- 一起吃饭、一起睡觉——用信赖而非恐惧
- 百年后成为调教界明星——确立「再调教」概念

### 五龙将——魔龙王的诞生
- 追查杀死克里斯塔的凶手——十年搜索无果——名声变为「无能混血儿」
- 人神出现告知拥有魔眼——可看到气息的痕迹
- 追踪到洞窟对战八大魔王之一——部下柯拉和斯克拉维战死——苦战十天
- 极限中解放腹中沸腾的能量——以自爆形式将整座山炸飞
- 萨列亚托救了坠落的拉普拉斯——醒来后将魔王绑缚——凯旋
- 被授予「魔龙王」称号与五龙将地位——「今后直属于我」
- 为朵拉的孩子起名「佩尔基乌斯」

### 龙之外交官——维系和平
- 随龙神参加众神会议——负责与其他世界的外交
- 在六个世界广交盟友——被视为维持六世界间和平的关键人物
- 龙界对魔族的敌意逐步减少

### 世界崩坏与真相
- 露娜莉亚被残杀——龙神开始复仇战争——毁灭四个世界
- 拉普拉斯见证世界崩坏——朵拉告知转移研究的真相
- 魔界崩坏后——在燃烧的魔神城遇到涅克罗斯拉克罗斯——得知双方都被陷害的真相
- 魔帝奇希莉斯卡里希斯被刺杀——龙族技术交换者失踪——嫁祸
- 拉普拉斯放走涅克罗斯拉克罗斯和魔神的孩子

### 龙界的终点——最后的使命
- 龙神命令「进攻人界」——五龙将决定阻止
- 五龙将对抗龙神——战斗六天——第七天龙神胜利——但被人神从背后偷袭
- 席拉德当场自杀谢罪——卡奥斯与马克士威为拖延龙界崩坏而死
- 朵拉临终请求照顾佩尔基乌斯——然后死去
- 龙神将孩子奥尔斯帝德送去未来——命令拉普拉斯「找出Hitogami的真相和打倒他的方法」
- 龙神与人神决战——七天七夜——双方消失

### 在人界的使命
- 作为最后的五龙将来到人界——寻找Hitogami所在和弱点
- 发现Hitogami被封印在「无的世界」
- 几千年间将龙族技术交给人族——回收改良后再研究
- 在世界各地建起遗迹——给五龙将后裔分发秘宝——设置多个保险
- **分裂**：第二次人魔大战——中了人神的计——与穿着斗神铠的巴迪冈迪战斗——被一分为二
- 分裂为「魔神拉普拉斯」（憎恨人族）与「技神拉普拉斯」（试图打倒神明）
- 失去全部记忆——再也记不起罗斯蒂娜

### 第25卷——遗产
- 巴迪叙述：世上真正不该与之为敌的始终只有一个——魔龙神拉普拉斯
- 制作了世界最强铠甲「斗神铠」
- 人类对「龙」的命名习惯源自拉普拉斯
- 第二次人魔大战中与巴迪同归于尽——被自己的铠甲打倒——讽刺的结局
- 大战爆炸摧毁大陆——产生林古斯海——大量死者化为魔力——生成魔神窟

## 能力演变

### 战斗/特殊能力
| 时期 | 能力 |
|------|------|
| 野生期 | 天生强大——单杀小山大的魔兽 |
| 五龙将期 | 魔眼——可看到气息痕迹；腹中沸腾能量（自爆级）；飞行（独特垂直方式） |
| 魔龙王期 | 被承认为世界最强龙族之一 |
| 人界期 | 魔龙神——货真价实的世界最强；发明斗神铠 |
| 分裂后 | 技神拉普拉斯 + 魔神拉普拉斯（各具不同能力） |

### 装备与持有物
- 斗神铠（发明者）——世界最强铠甲
- 龙界技术——武术、魔术、武器制造、遗迹建造

## 关键人际关系演变

### 龙神（养父/主君）
- 龙神救其性命——「你也算是我的孩子」
- 对龙神绝对忠诚——直到目睹龙神复仇毁灭
- 临终受托照顾奥尔斯帝德和寻找打倒人神的方法

### 露娜莉亚（养母）
- 将露娜莉亚当作自己的母亲
- 深受其养育之恩——她的死是龙神复仇的起点

### 朵拉（甲龙王·教官/同僚）
- 朵拉担任教官——从打斗到互信
- 为朵拉的孩子起名「佩尔基乌斯」
- 朵拉临终请求照顾佩尔基乌斯

### 萨列亚托（赤龙·伙伴）
- 第一只调教的龙——一生只亲近拉普拉斯一人
- 后成为赤龙王——在最关键时刻救了他

### 涅克罗斯拉克罗斯（昔日驱逐他的魔王）
- 曾暴力驱逐拉普拉斯离开城镇
- 多年后在世界崩溃中发现双方都被陷害——放走对方和魔神的孩子

### 巴迪冈迪
- 第二次人魔大战中穿着斗神铠与之战斗
- 拉普拉斯被斗神铠的奥义一分为二——分裂为技神与魔神

### 佩尔基乌斯
- 朵拉的孩子——拉普拉斯为其起名
- 临终受托照顾——遗言信：「请告诉那个孩子他的名字——他真正的名字是佩尔基乌斯」

### 奥尔斯帝德
- 龙神的儿子——拉普拉斯为之铺设整个人界的道路
- 讽刺：奥尔斯帝德为了去往人神之处，不得不杀死拉普拉斯取回秘宝

## 重要经历时间线
- **数万年前**：诞生于魔界，孤独长大
- **远古**：被龙神救回龙界，被露娜莉亚接纳为家人
- **龙界修行期**：拜朵拉为师，数十年完成龙族训练
- **龙之调教师期**：驯服萨列亚托，百年间成为调教界明星
- **五龙将就任**：追查杀死克里斯塔的凶手，击败八大魔王之一，获封「魔龙王」
- **龙界外交官期**：在六个世界广交盟友，维系和平
- **露娜莉亚之死**：龙神开始复仇，四个世界毁灭
- **龙界终点**：五龙将阻止龙神失效——全灭；受托使命
- **人界期（数千年）**：铺设道路、建遗迹、分发秘宝
- **第二次人魔大战**：与巴迪冈迪（斗神铠）同归于尽；分裂为技神与魔神
- **第25卷**：作为传说被提及——遗产持续影响世界
