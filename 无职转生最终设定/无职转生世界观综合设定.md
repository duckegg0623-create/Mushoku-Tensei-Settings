## 一、创世神话与宇宙观

### 1.1 创造神与世界的诞生

创造神活过悠久时间，创造了许多世界。晚年力衰时，最后的工作是创造出六个各自歪曲、不平衡的世界。创造神将六个世界连接取得平衡，制造七个分身（各世界的神），然后神死了。

### 1.2 七世界与七神

| 世界 | 神 | 特征 |
|------|-----|------|
| 龙的世界 | 龙神 | 天地反转，龙族居住 |
| 魔的世界 | 魔神 | 遍布毒与瘴气 |
| 兽的世界 | 兽神 | 苍郁森林与山脉 |
| 海的世界 | 海神 | 生命丰富的海 |
| 天的世界 | 天神 | 岩石浮于空中 |
| 人的世界 | 人神 | 平地与草原 |
| 无的世界 | 无神 | 什么都没有，白色地面无限延伸 |

### 1.3 世界的力量均衡法则

六个世界的力量有保持均衡的性质。如果某个世界的力量大幅减少，会像要填补空缺一样从别的世界吸收力量——连人都可能被吸过去。这就是**大转移事件**的根本成因。力量浓度越高，生物吸收过量力量就越可能变异为魔物。

召唤魔术的本质是从其他世界召唤生物，转移正相反，是把生物送到其他世界。两者本质相同，是最初期的召唤魔术。

## 二、六面世界

### 2.1 六个世界详情

**龙的世界**：天地反转，天是山、地是天空。龙鸣山凯欧斯（Chaos）是龙界最大最高的山，挖空山体建造的首都。有122个龙族城镇。天空中有不安全的区域，山脚有巨大飞龙栖息。

**魔的世界**：遍地毒沼泽、瘴气的雾、皲裂赤茶色地面。魔界中心是魔神都市塔列克与魔神城盖列克，位于巨大陨石坑中。

**兽的世界**：苍郁森林和山连绵不绝，每走一步都有虫和爬虫类出现。

**海的世界**：海界崩坏后不再是能住人的世界，但还有鱼等小生物。

**天的世界**：地面上积聚盐分，被水覆盖（水深至脚踝）。生物全在空中，基本被羽毛覆盖。岩石浮在空中。

**人的世界**：平地及草原。人类文明最发达。

**无的世界**：六面世界的中心部位。什么都没有，类似精神时光屋，只有白色地面无限延伸。人神被封印于此。

### 2.2 世界崩坏的顺序

1. **兽界**：龙神杀死兽神（撕成两半）→ 破坏九成 → 自我崩坏 → 人神接收幸存难民至人界 → 兽族死亡九成
2. **海界**：海族集结全军 → 海神对龙神用毒 → 龙界胜 → 海界崩坏
3. **天界**：天族集结遮蔽太阳的军队 → 天神极光烧瞎龙神一只眼 → 龙界胜 → 天界崩坏
4. **魔界**：最苛烈的战争，八大魔王 vs 五龙将 → 魔神巨大化 → 龙神用巨型魔法阵消灭魔神 → 魔界崩坏
5. **龙界**：五龙将阻止龙神攻人界 → 内战 → 人神偷袭龙神取出神玉捏碎 → 龙界崩坏 → 龙神临终送奥尔斯帝德去未来

最终只剩人的世界（即现在的世界），六个世界凝缩成一个，变为不平衡但又平衡的世界。

### 2.3 诸神形象

| 神 | 外貌描述 |
|----|----------|
| 海神 | 从口中生出大量触手，拥有像乌贼一样滑溜溜皮肤的男人 |
| 兽神 | 有着狗和猫两个头，骑在白狼上的男人 |
| 天神 | 额头上有两个眼睛，背上生着六个翅膀的美丽的男人 |
| 魔神 | 有八只手，六根角，身高三米的男人 |

## 五、种族设定

### 5.1 人族（Human）

贫弱身体、优秀头脑。擅长发明创造，将魔术系统化。寿命最短但文明最发达。

### 5.2 龙族（Dragon Tribe）

**社会与文化**：
- 龙神为顶点，五龙将辅佐，氏族制
- 绝对忠诚于龙神，龙神即正确
- 战死最光荣，病/老死不光彩（龙神改变了这一观念）
- 最敬礼：双拳在胸前交叉，叠起翅膀，对死者不握拳

**繁殖**：几千年才一次繁殖期，卵生，怀孕50年+孵化50年。产卵场集中管理。

**力量体系**：
- **龙气**：龙族内在的力量，可提升身体能力数倍、硬化皮肤，是所有种族中最强的力量
- **爪术**：将龙气注入锐爪切开对手
- **牙术**：将龙气注入拳头击打对手（拉普拉斯最擅长）
- **翼术**：将龙气注入翅膀进行剧烈加速减速
- **飞行**：靠翅膀生成反重力力场，利用风力滑行，龙族孩子需花上百年学会飞行

**龙种分类**：
| 龙种 | 特征 |
|------|------|
| 赤龙 | 体形较小，持续飞行距离长，凶暴，龙族主食与载具。需要长坡或悬崖才能起飞。一只可换阿斯拉金币一百枚 |
| 青龙 | 飞得较低，生活在云层下，每十年一次在青龙山脉产卵和育儿 |
| 地龙 | 生活于山谷 |
| 金龙 | 极强，龙将克里斯塔死于金龙巢附近 |
| 王龙 | 稀有品种，能操控重力 |
| 黑龙 | 稀有品种 |
| 白龙 | 稀有品种，古代白龙皮可制作防具 |

### 5.3 魔族（Demon Tribe）

上次人魔大战中属魔族阵营的种族总称，包括众多亚种。

#### 斯佩路德族（Superd）

- **特征**：翠绿头发、额头有红色宝石状物体（第三只眼/魔力器官，可观魔力流动）、个个美形
- **出身**：巴比诺斯地方
- **寿命**：约1000~2000年，与长耳族同等
- **尾巴/枪**：出生时会长有一根三叉尾巴，随成长变长，到一定年龄硬化并与身体分离。分离后的枪仍为身体一部分，越使用越锐利，绝对不会折断，可以贯穿世上万物
- **历史悲剧**：原本是守护世界的正义部族。五百年前魔神拉普拉斯将"魔枪"交给斯佩路德族，导致陷入疯狂自相残杀。拉普拉斯趁机屠杀其他种族并嫁祸。几乎被灭族
- **现状**：被世人视为"恶魔""死神"，被称为"绿发恶魔""Dead End"。在战争中放纵闹事，被敌我双方厌恶，战后被赶出魔大陆。对人族的迫害记忆犹新，尤其对米里斯教徒恐惧和仇恨
- **能力**：第三只眼可感知魔力流动和入侵者。只有斯佩路德族能识破魔神拉普拉斯的弱点
- **村落**：位于毕黑利尔王国"有去无回之森"深处，从事农业。对米里斯教团恐惧根深蒂固

#### 米格路德族（Migurd）

- **特征**：蓝色头发、额头有第三只眼（用于感应魔力/念话）、身材娇小
- **寿命**：约150~200岁。约十岁上下成长到成人模样，之后外貌不变直到150岁
- **能力**：念话（telepathy），即使分处两地也可与同族对话
- **习性**：不会对同族使用念话以外的方式对话
- **村落**：位于魔大陆，十几间屋子聚集，简陋栅栏，小田地。穷困至极
- **与斯佩路德族的关系**：能力相性极好，一个是通信兵（米格路德），一个是雷达兵（斯佩路德）
- **命名习惯**：常用Ro音开头

#### 鬼族（Ogre Tribe）

- **特征**：额头上长角，红黑色头发
- **首领**：被称为"鬼神"的强大武人
- **地位**：魔族之一，但未参加人魔大战和拉普拉斯战役，因此不被视为魔族
- **居住地**：第三都市黑雷鲁尔附近的鬼岛（火山岛）
- **义务**：有保护国家的义务

#### 不死魔族

- 不会死亡的天性，没有天敌，生存时间漫长。不回首过去，不留文献记录
- 不死魔王阿托菲拉托菲：魔大陆恐怖的象征，暴力之化身。第一次人魔大战结束时出生
- 涅克罗斯拉克罗斯：第一次人魔大战五大魔王之一，实力出类拔萃

### 5.4 兽族（Beast Tribe）

- 身体部分特征与哺乳动物相似，各种族拥有特殊五感
- 有发情期，对规定特别忠实

**德路迪亚族**（王族/正系，猫系）：

### 5.5 海族

- 支配全世界海洋，限制陆地居民海上通行
- 海族船只由海中魔兽拖曳，比人族船只速度更快更安全
- 海族寿命个体差很大
- 两族在内海和外海地盘分开，经常战争

## 六、魔术体系

### 6.1 三大类魔术

1. **攻击魔术**：用于战斗，分为火、水、风、土四大系统
2. **治愈魔术**：分为治疗、结界、解毒、神击四个系统
3. **召唤魔术**：通过魔法阵召唤魔兽、精怪魂灵等

### 6.2 七级位阶

初级 → 中级 → 上级 → 圣级 → 王级 → 帝级 → 神级

称呼方式：魔术师加"级"字（如水圣级魔术师），剑士不加（如水圣、剑圣）。

### 6.3 咏唱 vs 无咏唱

**咏唱流程**：产生 → 设定大小 → 设定射出速度 → 发动（蓄力时间上限1秒）

**无咏唱流程**：产生 → 设定大小（时间限制∞）→ 设定射出速度（时间限制∞）。可注入所有魔力。

| 阶段 | 咏唱发动（上级） | 无咏唱发动（初级，等量魔力） |
|------|------------------|---------------------------|
| 范围 | 50 | 1 |
| 威力 | 50~60 | 100 |
| 消耗魔力 | 100~ | 100 |
| 发动时间 | 11秒（咏唱10秒+蓄力1秒） | 11秒（蓄力10秒） |

> 初级咏唱：范围1，威力1~10，消耗魔力2~，发动时间2秒

- 无咏唱可通过训练增加1秒内投入的魔力
- 太习惯于咏唱魔术就不太能做到无咏唱
- **魔力成长黄金期**：五岁左右开始使用魔术可以让魔力总量爆炸性增加，超过十岁后魔力几乎不再成长

### 6.4 各系统详解

**攻击魔术**：
- 火圣级「闪光炎」（Flash Over）：瞬间将火焰以极大范围扩散，利用光线产生热能
- 水王级「雷光」（Lightning）：在空中展开魔力→压缩→击向指定场所。威力为已知最高
- 派生魔术「电击」（Electric）：小型豪雷积雨云压缩放出，麻痹对手神经
- 土魔术：鲁迪乌斯的知识不足无法分清楚各种金属。岩炮弹威力估计相当于舰炮射击（帝级参考）

**治愈魔术**：
- 中级足以应对全身烧伤、内脏破裂和骨折，全部无后遗症
- 神级治愈可以在死亡后没有经过太长时间的情况下复活。但脑袋被砍断或头部被压烂无法治好
- 神级解毒咏唱魔法能治疗的只有魔石病

**解毒魔术**：
- 初级：对应绝大多数症状
- 中级：需记50句以上咏唱
- 上级：需记上百句咏唱
- 圣级：背诵必要性降低，魔力消耗增加
- 王级以上：被各国研究并藏匿
- 神级：可治魔石病，咏唱存放于米里希昂大圣堂

**召唤魔术**：
- 分为「魔兽召唤」（唤出目前存在于某处的生物，不包括人）和「精灵召唤」（制作出具有某种能力的精灵）
- 无法召唤"活生生的人类"，但理论上只召唤"魂魄"存在可能性
- 物品召唤技术本身存在
- 本质是使用魔力从与世界或无界连接的其他世界召唤出什么，而非召唤素材构筑

**结界魔术**：
- 分为对物理型与对魔法型
- 圣级以上能设置兼具双方特性的结界
- 用途：保护自身、关住对象等

**神击魔术**：对鬼魂类和没有实体的气体型魔物特别有效。为对抗死灵魔术而诞生。战后逐渐衰退但仍流传。

**死灵魔术**：复活死者并加以操控的禁术。至今仍以骷髅兵、幽魂、会动的铠甲等魔物形态残留。

**飞行魔术**：存在但没写在教科书中，是某个种族的固有魔术。需要足以违抗重力的上升气流，战斗中使用需要相当锻炼。

### 6.5 魔法阵

- 可以做到远比咏唱精密的事情
- 将魔力结晶捣成粉状，混合特定材料制成涂料后绘制，使用一次后涂料蒸发
- 王级以上大规模魔术需特殊涂料，费用相当于国家预算
- 现代魔法阵知识大量失传，无法制作新的魔法阵
- 刻版可印刷魔法阵量产，不必手绘
- 世界上几乎所有魔术都能以魔法阵重现，但发动方式特殊的固有魔术不适用

**多重构造魔法阵**：将多套魔法阵重叠产生一个效果（来自自动人偶技术）

**立体魔法阵**：不拘泥于平面，利用纸张厚度层叠

### 6.6 转移魔法阵

- 必须同时满足**场所、装置、魔法阵**三项条件才能制作，需两人同时启动
- 双向转移魔法阵放上物体的瞬间会"交换"两边物品
- 成对的卷轴可作为紧急逃脱用转移魔法阵
- 残留在人魔大战时代的遗迹中，属于禁忌知识
- 鲁迪死后，转移魔法阵被设置在各国显眼场所

### 6.7 混合魔术

将魔术"按顺序使用"所带来的结果，与将不同系统魔术"同时使用"有差异。理论上三种混合可行，但鲁迪通过左右手控制魔术，所以很困难。

### 6.8 魔杖与魔力

- 魔杖的作用是**降低魔力消耗**而非增加威力上限。例：鲁迪双手最大威力为10，魔杖维持上限10但魔力消耗变为1/3
- 魔力消耗规律：同样魔术，变得越大耗魔越多，变得越小也耗魔越多
- 不同系统同时使用消耗更多魔力
- 可从自身以外的来源（如含有魔力的物质）抽取魔力

### 6.9 魔术教育

- 拉诺亚魔法大学：世界最大，学生超万人，七年制（最长九年），无论种族、人种、身分皆可入学
- 各国大城市有魔术学校
- 魔术公会对魔术师提供全面支援，最低F级，魔法大学毕业即D级
- 二十人中约一人有战斗能力，四百人中约一名魔术师

## 七、剑术体系

### 7.1 三大流派

| | 剑神流 | 水神流 | 北神流 |
|------|--------|--------|--------|
| **风格** | 攻击型 | 防御型 | 兵法型 |
| **核心** | 攻击正是最佳防御，制敌机先一击必杀 | 顺势卸招与反击，一对一最强 | 临机应变，适合佣兵与冒险者 |
| **装备** | 常穿便装（不依赖重量） | 常着沉重铠甲 | 多样化 |
| **根据地** | 剑之圣地（北方大地最西端） | 阿斯拉王国王都 | 王龙王国（中央大陆南部） |
| **奥义** | 「光之太刀」（剑速达光速） | 五种奥义，「剥夺剑界」 | 「胧十文字」「月光两断」 |

### 7.2 七级位阶

初级 → 中级 → 上级 → 圣级 → 王级 → 帝级 → 神级

- 上位剑士认真时能以50km/h速度奔跑
- 擅长学习的秀才类型能更快成为剑圣，但剑王以上需要是"感觉派"
- 剑之圣地约有20位剑圣，世界各地可能约100人
- 剑帝各流派不到5人，北神流王级不到50人
- 世界上有20位左右的神级人物（含魔术师）

### 7.3 斗气

- 基于自我意志掌控"斗气"就能成为上级剑士，完全驾驭即为剑圣
- **魔力总量 ≠ 斗气强度**——不是放出/消耗魔力，但需要一定程度的魔力
- 有人无论怎么锻炼也无法使用（如鲁迪乌斯）
- 战士拥有怪物般的体力是因为斗气
- 将魔力缠绕在构成身体的每一片肉上并加以固定，通过锻炼身体自然领悟
- 庞大的魔力若无相应肉体强度会摧毁自身

### 7.4 水神流详解

**五种奥义**：初代水神全部掌握，历代水神最多习得四种。

**剥夺剑界**：将前后左右上下360度全纳入攻击范围，对手一动即斩。无法斩杀完全没动的对手。射程范围约宴会会场大小。

水神流会直接将魔力架开。以微秒时间差几乎同时发射的攻击对其相当有效，但对水王以上对手无意义。对手越强，修行效果越大。

### 7.5 北神流详解

- 与其说是剑术更接近兵法，登峰造极可出奇制胜
- 有阿托菲派：不使用奇招，擅于一对多战斗的实战派
- 有奇诡派：使用各种道具（瞬间接着剂珠子、催泪弹、油+火焰剑、钩爪、迷彩披风、毒苦无等）
- 北神三剑士（第17卷）：北帝奥贝尔（"孔雀剑"）、北王维·塔（"光与暗"）、北王纳库尔贾德（双剑，米尔泰特族兔耳双胞胎）

### 7.6 剑神流详解

- 追求合理性的流派
- 独特礼法（单膝跪地）
- 剑术进步后越发不依赖重量
- 剑神加尔·法利昂：剑之圣地现任道场主人，随性之人

## 九、道具、装备与技术

### 9.1 魔道具 vs 魔力附加品

| | 魔道具 | 魔力附加品 |
|------|--------|-----------|
| 原理 | 本体刻有魔法阵，使用者咏唱灌注魔力发动 | 灌注魔力至物品内获得特殊能力 |
| 使用 | 可无限使用但消耗魔力 | 有使用次数限制但不消耗魔力 |
| 来源 | 人工产物 | 多从迷宫挖掘 |
| 示例 | 魔照石、通讯石板 | 疾风之靴、烦热斗篷 |

### 9.2 魔眼

| 魔眼 | 效果 |
|------|------|
| 魔力眼 | 直接看到魔力流动，每约一万人中一人拥有 |
| 识别眼 | 注视物体后看出详细情报，限于使用者所知 |
| 透视眼 | 看穿墙壁等物体，生物和具高浓度魔力的部分无法透视 |
| 千里眼 | 看到远处，难对焦，只能旁观无法出手 |
| 预知眼 | 看到几秒后的未来，难对焦但推荐 |
| 吸魔眼 | 吸收魔力，连自己施展的魔术也会被吸收 |
| 空绝眼 | 死神蓝道夫的魔眼，可张开屏障 |

### 9.3 知名装备

| 装备 | 效果 |
|------|------|
| 疾风之靴 | 奔跑速度提高数倍 |
| 烦热斗篷 | 隔绝热量，维持一定温度 |
| 压倒之手套 | 手掌所受冲击减半 |
| 斩铁剑 | 轻易斩断铁盾 |
| 凤雅龙剑 | 剑神七剑之一，可轻减对手斗气防御 |
| 魔剑"指折" | 王龙王骨头打造，价值十万阿斯拉金币 |
| 龙神手环 | 附加干扰人神窥视的效果 |

### 9.4 魔导铠

**发展史**：一式 → 一式改 → 二式 → 二式改 → 三式（开发中）→ 零式

| 型号 | 特征 |
|------|------|
| 一式（MK-1） | 高约三米，时速可达200公里，跳跃数百米。以札里夫义手覆盖全身的外部装甲 |
| 二式 | 仅手脚零件，小型化成功但出力受限 |
| 二式改 | 加装身体零件辅助魔法阵，性能匹敌圣级剑士 |
| 零式 | 高三米，藏青涂装。短期决战型，以极高魔力消耗换取大幅机动性和防御力 |
| 莉莉·格雷拉特版（483年） | 泛用魔导铠，降低风险 |

### 9.5 斗神铠

- 魔龙神拉普拉斯打造的最强铠甲，金色光辉，流线型设计
- 一旦穿上赋予足打倒神的力量，但强迫穿着者燃起斗争心，会逐渐丧失自我
- 直到着装者生命力耗尽无法脱下，以着装者魔力为动力
- 拉普拉斯在第二次人魔大战决战时也未曾穿上
- 巴迪在4200年前穿上斗神铠→大陆形状改变→创造出林古斯海
- 击败方法：用超越防御极限的高出力魔力消灭肉身；或用无视防御的魔剑切开并封印

### 9.6 其他技术与发明

**鲁迪乌斯发明**：电风扇、吸尘器、防水用品、保冷箱等（能普及的少——功能可通过魔术重现且材料特殊）

**魔力恢复剂**：成功做出，大热门商品（对奥尔斯帝德无效）

**通讯石板**：奥尔斯帝德制作，基于龙族魔道具原理（与七大列强石碑、冒险者公会卡片同源）

**札里夫义手**：土魔术驱动，有触觉反馈，可注入魔力增强力量

**自动人偶技术**：鲁迪乌斯与札诺巴耗费近十年研究。人工肉以土魔法粘土为基础，加入赤龙鳞等材料，半永久耐久性

**抗魔砖**：制造和贩卖由魔术公会独占，阿斯拉王国只有王都使用

## 十、各国地理与政治

### 10.1 阿斯拉王国

**定位**：全世界最大国家，掌握世界最大权力。中央大陆人类王国。

**王都亚尔斯**：世界最大都市，以人族英雄命名。银之宫坐落山丘上。五道城墙结构。

**四大领主**：
| 家族 | 特征 |
|------|------|
| 诺托斯·格雷拉特 | 保罗出身家族，男性遗传性地偏好巨乳 |
| 伯雷亚斯·格雷拉特 | 菲托亚领地领主，男性偏好"野兽"。传统：所有男孩由下任当家养育 |
| 艾乌洛斯·格雷拉特 | 另一分支 |
| 泽费洛斯·格雷拉特 | 另一分支 |

**王位争夺（爱丽儿时代）**：
- 第一王子格拉维尔派（有大臣大流士撑腰）——战力比6
- 第二王子哈尔法斯派——战力比3
- 第二公主爱丽儿派——战力比1（但有奥尔斯帝德和佩尔基乌斯后盾）

**地势**：
- 赤龙山脉：中央大陆山脉，赤龙栖息地
- 赤龙上颚：连结阿斯拉和北方大地的溪谷，宽可容大型马车会车
- 赤龙下颚：鲁迪差点被奥尔斯帝德杀死之地
- 龙须：北部国境与赤龙上颚之间的森林，有万里长城式城墙
- 阿尔堤尔河：流经王都旁，上建大桥

**政治体制**：贵族投票表决制，在宴会上当面表态。骑士相当于下级贵族。

### 10.2 米里斯神圣国

**首都米里希昂**：世界最大宗教——米里斯教团的根据地。

**四大区域**：
| 区域 | 特征 |

| 冒险者区 | 冒险者公会总部，各种面向冒险者的店铺，有贫民窟和赌场 |

| 神圣区 | 教团中枢，孤儿院，大圣堂，墓地，圣骑士团总部 |

**地标**：格兰湖中央纯白色"白之宫"、金色大圣堂、银色冒险者公会总部、七座高塔。

**三大骑士团**：
| 骑士团 | 装扮 | 职能 |

### 10.3 魔法三大国

拉诺亚王国、巴榭兰特公国、涅里斯公国。三国同盟，综合国力世界第四。

**魔法都市夏利亚**：
- 位于拉诺亚王国，接近国境
- 中央：最新式抗魔砖建造的魔术公会
- 东侧：学生区（魔法大学）
- 西侧：工房区（魔道具工房）
- 北侧：商业区（商业公会、奴隶市场）
- 南侧：旅馆区
- 布局参考米里希昂

**拉诺亚魔法大学**：世界最大级学校，学生超万人。

### 10.4 王龙王国

- 位于中央大陆南部，崇尚实力的帝国主义国家
- **首都维邦**：杂乱无章却充满活力。三大流派的道场相邻

**魔物强度分布**：魔大陆 = 北卡利特大陆 > 中央大陆北部 = 大森林 > 中央大陆南部 > 米里斯大陆 > 中央大陆西部

### 10.6 北方大地

**巴榭兰特公国**：禁卫骑士团使用雪龙兽鳞片铠甲和盾牌。第二都市罗森堡是北方大地入口。

**毕黑利尔王国**：位于中央大陆北部东端，三国绕。首都毕黑利尔（木造建筑多），第二都市伊雷尔，第三都市黑雷鲁尔（盛行渔业）。

**剑之圣地**：北方大地最西端。一年四季被白雪覆盖，环境非常严酷。

### 10.7 贝卡利特大陆

- 别名"魔大陆西南方"
- 魔物密度高，东部与北部意外安全
- 极端气候：灼热白天，酷寒黑夜。几乎没有绿洲
- 沙漠和密林比邻而居的奇特大陆

**迷宫都市拉庞**：周围存在无数迷宫，魔力附加品的产地。

**沙漠战士文化**：原住民战士文化。别名在首次踏上旅途时由族长命名。无结婚制度，孩子由全村共育。男孩首次射精、女孩初潮后行成人仪式离开村子成为战士。

### 10.8 天大陆

- 位于最北端，连接中央大陆与魔大陆的唯一大陆
- 标高约三千公尺断崖绝壁，天族（有翅膀人种）居住
- 一望无际宽广平原，没有一棵树，地面覆盖短小杂草及青苔
- 世界三大迷宫之一"地狱"所在地

### 10.9 其他地区

**大森林**（米里斯大陆）：兽族主要居住地。德路迪亚族对各族有巨大影响力，拥有能全面战争的战力。

**西隆王国**：中央大陆南部偏东小国。首都拉塔基亚。

**马尔齐渊佣兵国**：中央大陆纷争地带小国，靠派遣佣兵为生。守护神为森之女神列奴（原型为基列奴）。

**西隆王国政变**：帕库斯（第六王子）在死神蓝道夫协助下发动政变夺取王位，最终跳楼自尽。

### 10.10 奴隶制度

- 中央大陆南部和米里斯大陆：勉强经营
- 北方大陆：几乎完全合法甚至奖励，是重要商业行为之一
- 奴隶商人不受法律或秩序保护，法律上无限接近灰色地带

### 10.11 货币与经济

- 阿斯拉金币/银币：一人月生活费约银币两枚
- 米里斯体系：王钞（五万）、将钞（一万）、金币（五千）、银币（一千）、大铜币（一百）、铜币（十）
- 贝卡利特使用"新萨"币，以物易物为主流

## 十一、迷宫

### 11.1 迷宫机制

- 由普通洞窟因魔力沉滞变异而成
- **魔力结晶**：位于最深处，是迷宫的力量来源，也是高价魔术触媒
- **守护者（Boss）**：保护魔力结晶的强敌
- **魔力附加品**：迷宫将魔力注入装备中形成，大部分是垃圾，偶尔有神级异能
- 魔力和冒险者尸骸让迷宫不断扩张变深

### 11.2 重要迷宫

| 迷宫 | 位置 | 特征 |
|------|------|------|
| 龙神孔 | 不明 | 推测2500层以上，一万年历史，已知最深 |
| 地狱 | 天大陆 | 世界三大迷宫之一，最深处栖息冥王毕塔 |
| 魔神窟 | 林古斯海 | 世界三大迷宫之一。第二次人魔大战魔力集结地。内部重现大战时期场景。最深处放置斗神铠 |
| 转移迷宫 | 贝卡利特大陆 | 塞妮丝被困之地。魔石多头龙（九头龙）镇守最深处 |
| 图书迷宫 | 魔大陆亥雷司地区 | 全世界所有书本的抄本存放地，由黏族魔王管理 |

### 11.3 魔石多头龙（九头龙/Manatite Hydra）

- 鳞片由魔石构成，可吸收魔力使魔法无效化
- 头部被砍断后可再生
- 弱点：切断后用火烧灼断口可阻止再生
- 适应力极强：会自行咬断烧灼后的断口吞食再生
- 身后巨大的魔力结晶中封着塞妮丝

**事务所**：位于魔法都市夏利亚郊外。平房+地下迷宫（20间房间，各有转移魔法阵）。有武器库、书库、训练设施。

**鲁德佣兵团**：爱夏创立，员工约50人，以兽族为主。黑色大衣制服。业务为保镳租赁契约。设有米里斯和阿斯拉分部。

设立转移魔法阵于：阿斯拉王国、米里斯神圣国、大森林、王龙王国、魔大陆南部。利用通讯石板进行联络。

## 十三、特殊概念与术语

### 13.1 Hitogami（人神/人神的冒充者）

- 真实身份不明：可能找到创造神尸体获得神力
- 目的：让神与神争斗，毁灭其他世界，成为唯一世界的唯一神
- 位于无之世界的中心，拥有强大的未来视与远视能力
- 被龙神临终报复重创，封印于无的世界
- **使徒系统**：通过操控某人来使未来产生变化，一次最多操控三人，人神在梦中将人的意识召唤至无之世界
- 手段：暗杀魔帝嫁祸龙族，煽动四世界对龙族开战，引诱五龙将反抗龙神
- 特点：无法看见与奥尔斯帝德有关的未来（龙神秘术更强力），一旦牵连奥尔斯帝德会看到错误未来

### 13.2 龙神的秘术与诅咒

**奥尔斯帝德的秘术**：牺牲魔力恢复力（恢复速度变慢千倍），换取以下能力：
- 跳脱人神的视野（人神看不见他）
- 获得粗略的未来视能力
- **核心能力**：无论何时何地死去都能保有记忆从最初重来（时空穿越）
  - 起始点：甲龙历330年冬，中央大陆北部无名森林
  - 期限：200年——超过或死亡便强制送回起点
  - 已轮回超过一百次（两万年以上）

**四个诅咒**（奥尔斯帝德身负）：
1. 被所有生物厌恶或恐惧
2. 让人神无法看见
3~4. 未知

诅咒核心在头部，佩戴头盔可缓和效果（但数百年来进展极小）。

### 13.3 拉普拉斯

**初代五龙将幸存者 → 第二代龙神 → 堕落为"魔龙王"** → 最终分裂为：
- **魔神拉普拉斯**（憎恨人族）→ 五百年前被封印
- **技神拉普拉斯**（憎恨人神）

拉普拉斯因子：鲁迪乌斯、希露菲叶特都拥有——因此魔力庞大。拉普拉斯在四百年前被封印前分散因子，灵魂送往未来。

**复活条件（原定）**：西隆共和国建立 → 奴隶商人伯尔特·马其顿斯崭露头角 → 其子孙诞生魔神拉普拉斯。帕库斯死亡导致共和国无法建立，复活地点变为不可预测。

### 13.4 转移事件的真相

**甲龙历500年**："再生的神子"少女莉莉亚经历了无数次重复人生，每次都绝望死去。某次人生中，王国从异世界召唤来少年筱原秋人，少女因此感受到活着的实感。少年在战争中被杀后，少女临死前用尽全力使用"改变过去的能力"触及过去。

甲龙历400年，菲托亚领地上空出现时空裂缝，试图召唤与少年有关联的存在。但让本不存在的人类在过去存在是不可能的。此时一名毫无关联的灵魂（鲁迪前世）穿过时空裂缝，进入濒死的婴儿体内。鲁迪的存在削弱了世界的抵抗力，使时空裂缝扩大，七星被召唤。少女以几乎失去全部能力的代价重生。

—— 鲁迪的转生并非任何人刻意安排，纯粹是蝴蝶效应与意外。

### 13.5 诅咒与神子/咒子

- **神子与咒子本质相同**：魔力异常导致的特殊能力
- 区别仅在于正面/负面影响
- 拉普拉斯将诅咒转移至道具/枪上的前例存在
- 诅咒无法解除（因此才叫诅咒），但通过物品转移的诅咒会随时间逐渐消逝
- 异世界人不受诅咒影响（鲁迪不觉得奥尔斯帝德可怕）
- **"无力之神子"**：能力是无效化其他神子/咒子能力，被发现即遭杀害

### 13.6 转生法

初代龙神创造的技法，可将灵魂送往未来侵占其他生命体进而复活。需先将"因子"注入几个人，经过数百数千世代后打造出与灵魂契合的容器。

- 奥尔斯帝德：通过转生法从约两千年前送往未来
- 佩尔基乌斯：也是通过转生法转移到现代的龙族之一
- 因子特征：高魔力与魔术素质、绿色头发、可能天生拥有魔眼

### 13.7 五龙将与五样秘宝

| 龙将 | 分工 | 状态 |
|------|------|------|
| 圣龙帝席拉德 | 总统领，龙士指挥，龙神代理 | 可能已死 |
| 冥龙王马克士威 | 魔物退治，冥龙队指挥 | 可能已死 |
| 狂龙王卡奥斯 | 武器制造，龙神刀等神具锻造 | 已死（奥尔斯帝德回收秘宝） |
| 刚龙王克里斯塔 | 召唤/转移研究 | 死于金龙巢附近 |
| 甲龙王朵拉 | 龙的调教→后负责召唤/转移研究 | 存活（佩尔基乌斯） |
| 魔龙王拉普拉斯 | 龙神辅佐/外交（末席） | 分裂为魔神与技神 |

打倒人神需五样秘宝，由五龙将分别持有。五龙将中最后只有拉普拉斯和转生后的佩尔基乌斯、奥尔斯帝德活到正传时代。

### 13.8 异世界人特性

- 七星静香（穿越者）：魔力总量为零，不会变老
- 鲁迪乌斯（转生者）：具有魔力且正常成长
- 异世界人不受此世界诅咒影响
- 对历史造成巨大变动可能被世界"排除"

### 13.9 其他术语

- **乱魔（Disturb Magic）**：奥尔斯帝德使用的魔术。在手部输出魔力即将成形前用其他魔力妨碍并打散。原理单纯、消耗魔力少但操作困难
- **吸魔石**：取自魔石多头龙鳞片。有正反面——从背面注入魔力，正面放出振动波分解魔力制造物
- **魔石**：越透明越大效果越好，有色魔石能增幅对应系统。与魔力结晶不同
- **魔枪**：魔神拉普拉斯制造/赋予的诅咒武器，持有者陷入疯狂
- **数字迷信**："七"为吉利数字（受七英雄、七世界影响），"六"为不吉利数字
- **毒品（幸福粉末）**：北方大陆东部和魔大陆部分地区流通。需圣级以上解毒魔术才能抑制戒断症状

### 13.10 婚姻与习俗

- **成人之巡礼**（拉诺亚王国）：王族成人后微服旅行半年考察国内现状
- **水神流"舍弃家姓"**：水神伴侣必须舍弃家姓，源自初代水神列达尔的传说
- **德路迪亚族雨季规定**：严禁用餐时间外进食
