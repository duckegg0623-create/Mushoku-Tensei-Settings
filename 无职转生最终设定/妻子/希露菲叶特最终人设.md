# 希露菲叶特·格雷拉特（Sylphiette Greyrat / シルフィエット・グレイラット）

## 基本信息
- **全名**：希露菲叶特·格雷拉特（Sylphiette Greyrat），婚前名希露菲叶特
- **通称**：希露菲（Sylphy）、白妈妈（White Mama，子女称呼）、沉默的菲兹（Silent Fitz，化名）
- **性别**：女（幼年被鲁迪误认为男性，后以菲兹身份女扮男装多年）
- **种族**：长耳族（半精灵）+ 四分一兽人族混血
- **年龄**：与鲁迪乌斯同年（初登场约5~6岁）
- **出生地**：阿斯拉王国菲托亚领地布耶纳村
- **父亲**：罗尔兹（半精灵，村中猎人/森林监视员，艾莉娜丽洁之子）
- **母亲**：人类（混有兽人血统）
- **祖母**：艾莉娜丽洁·杜拉冈罗德（长耳族女战士，父：罗尔兹的生母）
- **配偶**：鲁迪乌斯·格雷拉特（第一夫人）
- **子女**：露西·格雷拉特（长女，发色像鲁迪）、齐格哈尔德（齐格）·格雷拉特（长子，绿发）
- **职业**：爱丽儿公主的守护术师（菲兹身份）→ 魔法大学学生/学生会成员 → 专职务家主妇
- **称号**：沉默的菲兹（Silent Fitz）
- **所属**：原爱丽儿公主随从、后为格雷拉特家第一夫人

## 外貌特征演变

### 童年期（第01卷，5~6岁）
- 翠绿色头发（非斯佩路德族，是混血返祖现象）、比父亲更长的尖耳
- 白瓷般肌肤，娃娃脸，极具美感，能引发保护欲
- 初期剪短发（为避免发色显眼、便于逃跑），后开始在鲁迪建议下留长
- 穿着简朴上衣和裤子（家境原因和便于逃跑）
- 被鲁迪误认为男孩

### 少年/少女期（第04卷，约10~11岁，菲兹身份）
- 银白色短发（因转移事件中魔力耗尽头发由绿变白）、墨镜遮掩眼睛
- 男装，被外界视为少年
- 右手戴白色手套（遮掩幼年被暖炉铁棍烫伤的疤痕）
- 外表忧郁——"沉默的菲兹"称号来源于极度沉默寡言且实力强悍
- 穿着爱丽儿赐予的疾风之靴、烦热斗篷、压倒之手套

### 青年期（第08~09卷，约15~16岁）
- 继续男装（菲兹身份），白色短发+墨镜+手套
- 身材娇小、扁平（长耳族特征），与安产型相去甚远（鲁迪视角）
- 第09卷末：摘掉墨镜表明真实身份后恢复女性装扮
- 头发已从超短发留长至短发

### 成年期（第10~26卷，16岁以后）
- 第10卷婚后：被告知停止男装，菲兹身份正式结束
- 怀孕期间体型变化，产后恢复良好——女人味大幅提升
- 母乳停后胸部恢复原状（被鲁迪调侃）
- 生了两个孩子后屁股依然很小
- 34岁（十年后）：看起仍像20岁前后——皮肤水嫩，外观变化缓慢（长耳族血统影响）
- 晚年（74岁鲁迪死时）：白发但仍精神——管理着整个大家庭，是格瑞拉特家不动的支柱

## 性格与心境演变

### 第01卷（约5~6岁）：内向的受霸凌少女
- 极其内向、胆怯、温顺
- 因绿发与危险的斯佩路德族相似，遭村中孩童霸凌——被丢泥巴、辱骂"魔族滚出去"
- 怀疑自己是否亲生孩子（头发和耳朵与父母都不同）
- 除鲁迪外没有朋友
- 不服输的一面：为学习魔术勤奋练习
- 对鲁迪言听计从
- 后逐渐展现坚强的意志——为"救出鲁迪"立志变强
- 对鲁迪感情显然超越"朋友"范畴
- 鲁迪得知她是女孩后态度骤变，一度造成她警觉疏远——在鲁迪展露"脆弱面"后恢复信赖

### 第04卷（约10~11岁，菲兹时期）：沉默的守护者
- 被转移至半空中（数百公尺高空），以无咏唱魔术紧急减速，以双脚骨折存活
- 魔力枯竭后失去意识，醒来时已失去一切
- 被爱丽儿看中能力而雇为守护术师——交换条件：帮忙寻找家人和老师
- 持续作高空坠落的恶梦、每晚呓语、尿床——被路克安慰
- 暗杀事件中徒手拦截飞向爱丽儿的小刀（中毒后自行施放解毒魔术）
- 贵族最初对他不满——实力被认可后"沉默的菲兹"称号确立
- 向路克学习宫中礼仪——每次被嘲笑后就苦练到完美

### 第08卷（约15岁）：双重身份的内疚与期待
- 入学考试认出鲁迪，但鲁迪未认出她——被说"初次见面"受到巨大打击
- 整整一年不敢表明身份——担心鲁迪已经完全忘记自己
- 作为"菲兹"与鲁迪建立友好关系——在图书馆跌倒事件、协助购买茱丽等场景中作为可靠学长出现
- 渴望与鲁迪重建关系，但不敢报上真名，不希望鲁迪成为爱丽儿的部下
- 嫉妒鲁迪与莉妮亚、普露塞娜的亲近
- 幻想着与鲁迪的婚姻生活（布耶纳村的屋子、双薪家庭、生孩子）
- 对鲁迪抱有恋爱感情——想建立"特别的关系"

### 第09卷（约16岁）：鼓起勇气——身份揭示
- 察觉到鲁迪对七星特别关注，产生嫉妒和焦虑——害怕鲁迪被人抢走
- 对自己不敢表明身份的怯懦感到自我厌恶
- 在爱丽儿策划的"森林之雨"作战中终于摘下太阳眼镜表明身份
- 坦承："我从以前就一直喜欢你。现在更喜欢了。"
- 从路克处获得媚药（原菲托亚领地产品）、爱丽儿资助零用钱买烈酒、换上爱丽儿挑选的新内衣
- 说出必杀台词"请慢用"后结合成功
- **核心转变**：从被动等待→主动争取——"如果你在紧要关头没有勇气，我将以公主之名禁止你再和鲁迪乌斯接触"（爱丽儿的激励）

### 第10卷（16~17岁）：新婚妻子
- 被告知停止男装——菲兹身份结束
- 与鲁迪正式结婚——负责家务、煮菜
- 新婚生活甜蜜——几乎每晚房事
- 醉酒后会撒娇、吃醋——曾嫉妒瑞杰路德
- 担心自己无法怀孕（长耳族血统浓厚不易怀孕）——提及鲁迪可以找小老婆
- 对鲁迪极度信任——照顾七星期间毫无怀疑
- 被揭晓为艾莉娜丽洁的孙女（罗尔兹之女）

### 第11卷（约17岁）：准母亲与留守
- **宣布怀孕**（约2个月）——鲁迪狂喜落泪
- 为鲁迪准备护身符（首次之夜的纪念物放在神龛）
- 鲁迪出发后在外人面前表现坚强，但诺伦看出她"心里满是辛酸苦楚"
- 怀着孩子独自等待丈夫归来

### 第12卷（约17岁）：接纳第二夫人
- 鲁迪带回洛琪希请求娶为第二夫人时——"她一直预感鲁迪终会带其他女人回来"
- 坦然接受——展示了作为第一夫人的包容力
- 女儿露西出生

### 第13卷（17~18岁）：第一夫人的成长
- 产后恢复良好，女人味大幅提升
- 坦言曾嫉妒洛琪希——"鲁迪提起洛琪希时总是充满崇敬"，但实际见到后发现"她只是爱着鲁迪的普通女孩，和我一样"
- 在洛琪希差点主动离开时挽留她
- 与鲁迪和洛琪希实现三人约会和同寝
- 允许鲁迪将来再带"特别之人"回家——条件是必须带来见她
- **展现威严**："我不打算束缚鲁迪，但无法原谅根本不喜欢鲁迪却想成为你妻子的女孩"
- 吃醋时会鼓起脸颊——可爱反应让鲁迪感动

### 第14~15卷（约18岁）：接纳第三夫人
- 空中要塞之行中首次造访——极度恐高
- 为洛琪希被挡在门外发声
- 未来鲁迪出现时在场——未来世界线中她和洛琪希都死了
- 鲁迪告知可能需要迎娶艾莉丝后——最初难以释怀（艾莉丝曾伤害过鲁迪），后理解接受
- 协助制作魔导铠——以无咏唱土魔术，"很开心被依靠"
- 鲁迪决战时——被鲁迪偷看独处时抱着枕头滚动——"你或许就快要有弟弟或妹妹了……！"
- 到场后发现鲁迪败北——做好舍命一战的觉悟："我要战斗！"
- **对艾莉丝的开导**："你拥有我们没有的力量——站在鲁迪身前的力量。希望你对自己更有自信。今后也请多多指教喔，艾莉丝。"——发言时"目光炯炯，完全没有退缩"

### 第16~17卷（约18~19岁）：菲兹的最后活跃
- 持续作为爱丽儿护卫（菲兹）——同时兼任家庭主妇
- 发起并主持"第二十六回格雷拉特家的例行会议"（女子会）
- 图书迷宫探险中与爱丽儿同行的中卫
- 阿斯拉政变时担任爱丽儿替身——承受每日暗杀者袭击
- 在宴会中执行魔术演出——「火柱」照亮空中要塞
- **路克失控事件中的名场面**：被路克质问是否信任奥尔斯帝德——"完全不信任""因为我相信鲁迪啊！"
- 被问"要我杀鲁迪的话你选哪边"——立即回答"我会选择鲁迪"
- "夫妻不就是要白头偕老，共患难吗？"
- 对路克的罕见威胁："如果路克丧失自我打算危害爱丽儿，我就会杀了路克"——鲁迪第一次觉得希露菲"这么可怕"
- **战后决定**：正式辞去爱丽儿护卫——决定以"母亲"的身份守护鲁迪的背后

### 第18~21卷（20~22岁）：专职务家主妇
- 第二胎怀孕
- 正式回归家庭——专心育儿，"鲁迪，今后就交给我吧"
- 鲁迪前往米里斯时留守——被全家人绝对信任："家里有希露菲在不会有事的"
- 被鲁迪评价为已"不再只是被我保护的存在"

### 第22卷（约23岁）：产后创伤与修复
- 第二胎分娩——婴儿头发为绿色（与昔日希露菲相同）
- **看到绿发后当场昏倒**——深层心理创伤被触发（幼年因绿发遭霸凌的记忆）
- 鲁迪忘记想好孩子名字——让她露出"被背叛的表情"
- 反复做噩梦——绿发小孩遭到黑影追赶
- 真正的困扰："比起齐格可能是拉普拉斯，更在意他的头发是绿色的"
- 一度心想"让佩尔基乌斯带走齐格说不定反而轻松"——随后为此自我厌恶
- **被洛琪希开导**："差别待遇这种事，我累积了不少经验！"
- 鲁迪回应："哪怕是击毁空中要塞，我也会保护齐格"——低声回"嗯"

### 第23卷（约24岁）：逐渐治愈
- 在天大陆的亚尔切镇——注意到"这里没有人以异样眼光看待齐格"
- "如果大家都能这样就好了……"
- 精神状态逐渐恢复——"语气很轻松"，脚程很快（用治愈魔术恢复产后体力）
- 以魔术与洛琪希共同攻击霸王腭龙

### 第24~25卷：家庭不动之支柱
- 通过古老转移遗迹前往剑之圣地——被绝对信任
- 通讯中断时艾莉丝坚信"鲁迪乌斯不在家时，家里由希露菲来守护"
- 在家中坚强守护孩子们

### 第26卷：战场上的治疗女神与晚年
- **关键时刻的援军**：与基列奴、伊佐卢缇一同作为援军赶到战场
- 在战线崩溃的千钧一发之际——如灵活的老鼠在战场奔走
- **以无咏唱治愈魔术一碰就治好伤员**——速度远超鲁迪和克里夫两人
- 一人包揽所有伤员治疗——迅速治好冬迦、札诺巴、艾莉丝与瑞杰路德，战线恢复
- 下达撤退命令："卢迪！这里由我们挡下——你快去村子！洛克希在等你！"
- 之后继续奔走在战场上治疗负伤者
- 十年后（34岁）：内在完全变成了"老妈"——变得唠叨
- 鲁迪74岁临终时在床前握住鲁迪的手——解释在场每个人的身份
- 晚年：白发但仍精神——管理着整个格雷拉特大家庭

### 六面世界3篇（40岁+）：白妈妈送别儿子
- 齐格无职宅家的一年里——每天叫他起床，不是休息日一定准备早餐
- 最先注意到齐格在躲父亲（在早饭桌上看着空座位叹气）
- 齐格做正义的伙伴晚归时——偶尔会起床迎接
- 第一个察觉儿子不对劲——但遵循"暂时放着比较好"的方针
- 齐格宣布要远行后——咬住嘴唇马上要哭出来
- 大姐露西已远嫁米里斯——齐格也要走的话两个亲生孩子都离家了
- 终于忍不住落泪，但哭着说："我很高兴啊——齐格能好好的走向下一步了"
- 红妈妈和蓝妈妈安抚她："要笑着目送他啊"
- 小时候苦口婆心教育齐格："结交朋友——不要欺负弱小"

## 能力演变

### 魔术
- **第01卷（5~6岁，经鲁迪教导）**：无咏唱魔术——可使用到中级；水系统、风系统擅长；火系统不擅长（幼时被暖炉铁棍烫伤的手掌有疤痕，心理阴影）；魔力上限经过训练大幅提升；是鲁迪之外极少数会无咏唱的人
- **第04卷（约10~11岁，菲兹时期）**：无咏唱上级魔术、全系统中级以上；以无咏唱魔术在高空坠落时紧急减速存活；使用"冲击波"和"爆音冲击波"对抗刺客；中毒后自行施放解毒魔术
- **第05卷**：无咏唱治愈魔术上级——鲁迪反而不会
- **第08卷（约15岁）**：无咏唱魔术师，擅长多种魔术；魔力总量高；从小开始使用魔术积累丰富
- **第09卷（约16岁）**：实力相当于圣级剑士；在冰雹森林与鲁迪争夺天候操控时可分庭抗礼
- **第10卷（约16~17岁）**：火魔术中级、水魔术上级、土魔术上级、风魔术上级、治愈魔术上级（无咏唱）、解毒魔术上级（无咏唱）；正在学习乱魔
- **第12卷以后**：可执行王级治愈魔术卷轴
- **第13卷**：向洛琪希学习水王级魔术"雷光"——成功使出"豪雷积雨云"但未能压缩为"雷光"（魔力耗尽），洛琪希安慰"我五次也会失败一次"
- **第26卷**：**无咏唱治愈魔术在战场上发挥决定性作用**——一碰就治好，速度远超鲁迪和克里夫两人，一人包揽所有伤员治疗——是之前治愈魔术伏笔的完美回收

### 战斗表现
- **第04卷**：独自一人击败二十名袭击者
- **第08卷**：被评价为"实力相当于圣级剑士"
- **第17卷**：阿斯拉政变中担任中卫魔术师——快速判断状况并对应，以漂亮走位战斗，比基列奴更快拔出魔杖

### 日常技能
- 家务（煮菜、打扫寝室等）——成为主妇后的日常
- 育儿——哺乳、教导说话、教育孩子
- 学习读写和算术（九九乘法、两位数乘法）——第01卷鲁迪教导
- 掌握基础理科知识——第01卷鲁迪教导

### 装备（作为菲兹时爱丽儿赐予）
- 疾风之靴：提高奔跑速度数倍
- 烦热斗篷：隔绝热量维持一定温度
- 压倒之手套：手掌所受冲击减半
- 腰间的魔杖：小型棒状（新手用级别）
- 太阳眼镜（魔力附加品，遮掩眼睛和身份）
- 白色手套（遮掩手掌烧伤疤痕）
- 婚后：物理防御戒指

## 关键人际关系演变

### 鲁迪乌斯·格雷拉特（青梅竹马 → 丈夫）
- **第01卷（5~6岁）**：鲁迪第一位朋友，被鲁迪从霸凌中拯救。日常在山丘大树下练习魔术和读书。在鲁迪说要离开村子时哭着紧抱不放手。得知鲁迪被强行带走时以无咏唱中级魔术攻击保罗（被化解），发誓"我会变强，强到能救出鲁迪"
- **第04~08卷**：远隔多年后以"菲兹"身份与鲁迪重逢——整整一年不敢表明身份，被说"初次见面"极度受伤
- **第09卷（16岁）**：在洞窟中摘下太阳眼镜表明身份——"我从以前就一直喜欢你。现在更喜欢了。" 与鲁迪结合，**ED痊愈**
- **第10卷**：正式结婚——"请慢用"
- **第12卷**：大度接受洛琪希为第二夫人——"一直预感鲁迪终会带其他女人回来"
- **第15卷**：接纳艾莉丝为第三夫人——做好舍命一战的觉悟
- **第17卷**：在阿斯拉政变时公开宣言选择鲁迪——"我会选择鲁迪"
- **第22~23卷**：因绿发婴儿事件夫妻关系短暂裂痕→逐渐修复
- **第26卷**：在战场上一碰就治好伤员——鲁迪的绝对后盾
- 相伴至鲁迪74岁临终——在床前握住他的手

### 爱丽儿·阿涅摩伊·阿斯拉（救命恩人 → 好友）
- **第04卷**：被爱丽儿从转移事件中救下——视为救命恩人，愿为其效力
- **第09卷**：爱丽儿策划"森林之雨"告白作战——"如果你在紧要关头没有勇气，我将以公主之名禁止你再和鲁迪乌斯接触"
- **第17卷**：战后正式辞去护卫，含泪告别——"帮助朋友是理所当然的啊"
- 长期互信关系——爱丽儿视希露菲为"朋友"而非部下

### 路克·诺托斯·格雷拉特（室友 → 奇妙的朋友）
- **第04卷**：室友兼同事，路克对他异常温柔
- 路克常调侃"没有胸部的女人没有魅力"，但也说"正因为你不像个女人，我才能当你的朋友"
- 第09卷：路克赠送昂贵媚药
- 第17卷：路克失控事件后道歉——"别在意，毕竟路克也感到很不安"
- 第16卷：对路克的罕见威胁——"如果路克丧失自我打算危害爱丽儿，我就会杀了路克"

### 艾莉娜丽洁·杜拉冈罗德（祖母）
- **第10卷**：被揭晓为艾莉娜丽洁的孙女（罗尔兹之女）
- 被告知"因长耳族血统浓厚，不易怀孕"

### 洛琪希·米格路德（嫉妒对象 → 亲密的共妻与开导者）
- **第13卷**：坦言曾嫉妒洛琪希（"鲁迪提起洛琪希时总是充满崇敬"），实际见到后发现"她只是爱着鲁迪的普通女孩"
- 和谐共处，共同照顾子女——两人互相尊重，洛琪希察觉鲁迪异常时首先找希露菲商量
- **第23卷**：被洛琪希开导——"差别待遇这种事，我累积了不少经验！"——帮助希露菲走出产后创伤
- "洛琪希真的很可靠"

### 艾莉丝·格雷拉特（释怀对象 → 姐妹般的关系）
- **第15卷**：最初对曾伤害鲁迪的艾莉丝有抗拒，理解后接受
- **对艾莉丝的关键开导**："你拥有我们没有的力量——站在鲁迪身前的力量。今后也请多多指教喔，艾莉丝。"
- 像对待不成材的妹妹般照顾艾莉丝
- 分娩时艾莉丝负责找医生——三人形成"前卫/后卫/留守"的互补分工

### 子女
- **露西·格雷拉特**：长女（发色像鲁迪）——远嫁米里斯
- **齐格哈尔德（齐格）·格雷拉特**：长子（绿发）——从产后心理创伤到完全接纳，"我很高兴啊——齐格能好好的走向下一步了"
- 对子女教育方针："结交朋友——不要欺负弱小"

## 重要经历时间线

- **第01卷（约5~6岁）**：因绿发遭霸凌 → 被鲁迪拯救成为第一位朋友 → 学习无咏唱魔术至中级 → 鲁迪发现她是女孩 → 得知鲁迪被强行带走后以无咏唱魔术攻击保罗 → 发誓"我会变强，强到能救出鲁迪"
- **菲托亚转移事件**：从数百米高空坠落，以无咏唱魔术紧急减速、双腿骨折存活，头发由绿变白 → 被爱丽儿收留
- **第04卷（约10~11岁）**：成为爱丽儿守护术师 → 刺客事件确立"沉默的菲兹"称号 → 收到爱丽儿赐予的四件魔道具 → 与路克成为室友
- **第06卷**：鲁迪在难民营失踪名单上确认她生存但下落不明
- **第08卷（15岁）**：入学魔法大学 → 入学考试认出鲁迪但被说"初次见面"受巨大打击 → 以"菲兹学长"身份与鲁迪建立友好关系 → 协助图书馆调查转移事件
- **第09卷（16岁）**：爱丽儿策划"森林之雨"作战 → 在洞窟中摘下太阳眼镜表明身份 → 告白 → 携带媚药夜访——"请慢用" → **ED痊愈**
- **第10卷（16~17岁）**：与鲁迪正式结婚 → 停止男装（菲兹身份结束）→ 新婚生活 → 被揭晓为艾莉娜丽洁的孙女 → 担心不易怀孕
- **第11卷（约17岁）**：**宣布怀孕（约2个月）** → 鲁迪外出时留守 → 怀着孩子独自等待
- **第12卷（约17岁）**：鲁迪带回洛琪希请求娶为第二夫人 → 坦然接受 → 女儿露西出生
- **第13卷（17~18岁）**：产后恢复 → 女人味提升 → 坦言曾嫉妒洛琪希 → 与鲁迪和洛琪希实现三人同寝 → **展现第一夫人威严**
- **第14卷（约18岁）**：空中要塞之行（极度恐高）→ 为洛琪希被挡在门外发声 → 与未来鲁迪乌斯见面——未来世界线中她和洛琪希都死了
- **第15卷（约18岁）**：接纳艾莉丝为第三夫人 → 协助魔导铠制作 → 开导艾莉丝 → 做好舍命一战的觉悟："我要战斗！"
- **第16卷（约18岁）**：发起并主持女子会 → 对路克的罕见威胁——鲁迪第一次觉得她"这么可怕" → 图书迷宫探险
- **第17卷（约18~19岁）**：阿斯拉政变担任爱丽儿替身 → **路克失控事件中宣言"我会选择鲁迪"** → 战后**正式辞去爱丽儿护卫** → 含泪与爱丽儿告别 → 决定以"母亲"身份守护鲁迪的背后
- **第18~19卷（20岁）**：第二胎怀孕 → 成为三个孩子的母亲
- **第20~21卷（22岁）**：留守夏利亚——被全家人绝对信任
- **第22卷（约23岁）**：**第二胎分娩——婴儿绿发，看到后当场昏倒** → 内心创伤暴露 → 对鲁迪感到失望（他忘记想名字）→ 被洛琪希开导 → 夫妻关系短暂裂痕
- **第23卷（约24岁）**：天大陆之旅中逐渐治愈 → 发现"这里没有人以异样眼光看待齐格"→ 精神状态恢复"语气很轻松"
- **第24~25卷**：通过古老遗迹前往剑之圣地 → 被全家人绝对信任 → 在家中坚强守护
- **第26卷（约24~34~74岁）**：**总力战中作为援军赶到——以无咏唱治愈魔术力挽狂澜** → 一人包揽所有伤员治疗 → 十年后成为"老妈"型人物 → 鲁迪临终时在床前握住他的手 → 晚年管理着整个格雷拉特家
