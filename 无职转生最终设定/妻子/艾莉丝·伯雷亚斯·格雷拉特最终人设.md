# 艾莉丝·伯雷亚斯·格雷拉特 / 艾莉丝·格雷拉特（Eris Boreas Greyrat / エリス・ボレアス・グレイラット）

## 基本信息
- **全名**：艾莉丝·伯雷亚斯·格雷拉特（Eris Boreas Greyrat）→ 婚后称艾莉丝·格雷拉特（舍弃伯雷亚斯之名）
- **通称**：艾莉丝（Eris）、红妈妈（Red Mama，子女称呼）、狂犬、狂剑王
- **性别**：女
- **种族**：人类
- **出生地**：阿斯拉王国菲托亚领地罗亚（伯雷亚斯·格雷拉特家）
- **家族地位**：阿斯拉王国四大领主之一伯雷亚斯家孙女 → 格雷拉特家第三夫人
- **祖父**：绍罗斯·伯雷亚斯·格雷拉特
- **父亲**：菲利普·伯雷亚斯·格雷拉特
- **母亲**：希尔达·伯雷亚斯·格雷拉特
- **兄弟**：有两个兄弟（哥哥和弟弟，均被詹姆士夺走为养子）
- **发色**：红色（伯雷亚斯家特征）
- **配偶**：鲁迪乌斯·格雷拉特（第三夫人）
- **子女**：亚尔斯·格雷拉特（长子）、克莉丝汀娜·格雷拉特（次女）
- **师傅**：基列奴·泰德路迪亚（剑神流，唯一承认的师傅）、瑞杰路德·斯佩路迪亚（战斗指导）、加尔·法利昂（剑神，修行对象但非师傅）
- **流派**：剑神流（剑王级，获免许皆传）、兼修北神流/魔族技术
- **称号**：剑圣（第09卷）→ 剑王/狂剑王（第14卷）

## 外貌特征演变

### 童年期（第02卷，约9~12岁）
- 红色长发（伯雷亚斯家特征），贵族大小姐
- 起初穿着邋遢（不懂礼仪），经鲁迪教导后学会打扮
- 十岁生日舞会上穿着正式礼服，表现如同淑女

### 少年/少女期（第03~06卷，12~15岁）
- 魔大陆旅行中为隐藏身份将头发染成其他颜色
- 穿着轻便盔甲，完全适应冒险者装扮
- 第06卷末：剪掉自己的红色长发，留下一束束红发在地上，标志着"舍弃伯雷亚斯之名"

### 青年期（第09~15卷，17~18岁）
- 短发红色，剑士装束——剑王大衣，佩戴两把剑
- 脸上有与北帝奥贝尔战斗留下的伤痕（第10卷）
- 身体经过长期修行变得极为结实
- 第15卷：穿上黑色连身裙、喷香水与鲁迪重逢

### 成年期（第16~26卷，18岁以后）
- 短发红发，身为战士和母亲的威严并存
- 哺乳期的神秘母亲形象——"那个艾莉丝成为母亲后充满神秘"
- 十年后（34岁）：皮肤年龄保持20岁后半——归功于不间断锻炼
- 老年期（70岁+）：仍精神地锻炼，某天倒在床上后再也没有醒来

## 性格与心境演变

### 第02卷（约9~12岁）：从野兽到淑女
- **初期**：极端暴力，被称为无法驯服的野兽、暴君；不会读写、不会算术、不懂礼仪
- 父母已对她绝望——菲利普认为她"不可能担任贵族"
- 第一次与鲁迪见面时直接拳打脚踢将其击倒
- **绑架事件转折**：被鲁迪所救后开始产生信任，逐渐听从鲁迪的教育
- 十岁以后：上课认真听讲，很少出手打人，努力练习剑术，假日也自发加练
- 开始在意自己在鲁迪面前的形象，学会打扮
- 在十岁生日舞会上表现如同淑女——让所有人大吃一惊
- 对鲁迪产生特殊感情（被基列奴观察到类似塞妮丝被保罗吸引时的样子）
- 十岁生日穿红色连身睡衣主动陪睡，给出"五年后"的承诺

### 第03~04卷（12~14岁）：魔大陆的女战士
- 转移后与鲁迪相依为命，面对突发状况表现出坚强意志
- 在"Dead End"队伍担任前卫，与鲁迪形成配合
- 在瑞杰路德指导下战斗技巧继续成长
- 对鲁迪以外的男性依然警戒心很强
- 学会魔神语（可吵架）；学会少量兽神语
- 在旅途中逐步学会控制暴力倾向——但对性骚扰仍会反击
- 守护走私儿童时"开心到笑容满面"（被鲁迪评价为格雷拉特血统）
- 与兽族女孩建立友谊——离别时流泪
- 为基列奴辩护——"基列奴是我的师父！是我最尊敬的人！"

### 第05卷（约12岁）：青春期少女
- 看到鲁迪因保罗消沉后极度愤怒——"我要去杀了那家伙"
- 对鲁迪极度依恋、自豪，视为最重要的人（滔滔不绝讲述鲁迪的事，"自豪表情宛如恋爱中的少女"）
- 被克里夫求婚——"绝对不要！"
- 勉强参加家族聚餐，对保罗保持警戒但能克制
- 吃醋时会满脸通红
- 首次实战杀人——依旧泰然自若
- 受瑞杰路德影响，看到小孩遇险会不顾一切出手相救

### 第06卷（15岁）：重大转折——离别
- 经瑞杰路德正式认定为"战士"（不再是孩子）
- 鲁迪被奥尔斯帝德杀死时崩溃哭喊求救，独自完成照顾
- 得知祖父被处决、双亲死于纷争地带后——以怒吼制止争吵，要求独处
- **与鲁迪初夜**：深夜穿黑色连身裙、喷香水造访——"我要你成为我的家人"
- **离别决定**：剪掉红色长发，留下纸条——"现在的我配不上鲁迪乌斯，所以我要去旅行"
- **离别的真正理由**：不是讨厌鲁迪，而是太爱他——怕自己会拖累他继续堕落，想等彼此再成长后重逢
- 前往剑之圣地修行——为成为配得上鲁迪的女性

### 第09~10卷（17岁）：剑之圣地的狂犬
- 初到剑之圣地：对剑神第一句话"我对你这样的杂碎没兴趣！"
- 目标：斩杀龙神奥尔斯帝德
- 突袭吉诺·布里兹一击打晕，对决妮娜·法利昂用拳头打烂对方的脸
- 即日被授予剑圣称号，成为剑神直属弟子
- 两年来专注接受锻炼——为鲁迪而努力变强
- 对敌人毫不留情、不善社交、沉默寡言——但提到鲁迪时会侃侃而谈
- 对自己的恋爱经历感到自豪又害羞
- 不擅长思考，通过反复挥剑来理解——手上水泡磨破用布随意包扎
- 被北帝奥贝尔击败后脸上留下伤痕，懊悔扭曲了表情

### 第13卷：技术的觉醒
- 与水王伊佐露缇交手——三十分钟内"死亡"近百次
- 弱点暴露：放出太多杀气（水神流对杀气极度敏感），不擅长抑制杀气
- 与妮娜、伊佐露缇形成三方僵局并共同训练
- 开始每天洗澡、好好吃饭——回想鲁迪时嘴角露笑，在这种状态下没有放出杀气
- 一年后与水王交手时终于略占上风

### 第14卷：剑王诞生
- 将"光之太刀"改造——牺牲威力换取速度
- 击败妮娜·法利昂——证明自己超越了单纯的蛮力
- 正式获封"剑王"称号和"免许皆传"
- 说出"我想回家了……想回到我的家人身边"——与鲁迪写信求助的时刻不谋而合

### 第15卷：归队与结婚
- 收到鲁迪信件后立即出发——"不快点的话会来不及呢"
- 在鲁迪家门口犹豫了一小时未敲门，被爱夏主动带进家中
- 在奥尔斯帝德战介入救场——使用剑神流第三阶段（二刀流+北神流掷剑技+"光之太刀"）
- 做好牺牲觉悟："你们快带鲁迪乌斯逃走！我就算牺牲性命也会拦住奥尔斯帝德"
- 在家中感到自卑——不会家事不会赚钱，相继尝试料理和警备工作均失败
- 被希露菲开导："你拥有我们没有的力量——站在鲁迪身前的力量"
- 主动表白："我现在也依旧喜欢鲁迪乌斯。我想被鲁迪乌斯爱着"
- 提出决斗："要是我赢了，你也要爱我"
- 当晚化为"野兽"主动压倒鲁迪——"鲁迪乌斯，我想生个男孩子"

### 第16~18卷：妻子与战士
- 在家中的日常：哄露西、抱雷欧（雷欧怕她）、参加女子会
- 模拟战碾压鲁迪：十战九胜——"由我来保护鲁迪乌斯！"
- 对怀孕态度："到时候再说啦！"——字典里没有家庭计划
- 被希露菲安排协助旅行准备
- 作为"勇者伙伴"参与阿托菲战——被毛茸茸使魔认定是"强者"而亲近
- 不擅长料理——在厨房烤肉烧成木炭
- 被鲁迪日记提及——怀上第三子（亚尔斯）

### 第19~21卷：母亲
- 因怀孕留守——鲁迪外出期间产下长子亚尔斯（红发）
- 作为三个孩子的母亲——新的家庭角色
- 被爱夏叫来米里斯保护鲁迪——全副武装潇洒登场："既然我来了就不要紧了！我会把所有人砍成两半！"
- 收神子为弟子——砍下巴尔塔树树枝削成木棒
- 指导神子挥剑："就能变得像我一样"

### 第22~25卷：狂剑王的全盛期
- 涅克罗斯要塞之战：独自面对阿托菲——"我根本不需要什么名誉。不过，我会砍下妳的脖子"
- 以光之太刀砍入阿托菲脖颈三分之一——被阿托菲以刺剑阻止
- 剑之圣地被称为"狂剑王"——路上剑士主动让路
- 斩杀前剑神加尔·法利昂——"成长了啊""已经熟练地掌握剑神流了"——斩杀后发出悲痛呼号
- 露西抱怨"红妈妈很粗鲁"（洗头技术不好）
- 对战亚历山大多次切入攻击

### 第26卷：最后的战斗与晚年
- 参加对斗神总力战——作为前排输出参战
- 斩击无法对斗神铠造成实质伤害——被斗神一拳打中口吐鲜血
- 被希露菲的无咏唱治愈魔术救回
- 十年后：生下克莉丝汀娜，性欲减弱但偶尔仍会袭击鲁迪。开始教导孩子们剑术——凶暴性降低，耐性变好。摸屁股或胸部仍会被揍
- 70岁后仍精神锻炼——某天跑完步倒在床上后再也没有醒来
- 鲁迪哭了三天——想"要是早点发现——早点施治愈魔法就能治好了"

### 蛇足篇：作为母亲的严格与慈爱
- 家庭会议中揍了爱夏——"明明教了你要守护的！"
- 与儿子亚尔斯死斗——以压倒性速度和力量攻击，砍伤肩膀、斩断手腕、空手殴打
- 最终放下架势："不愧是鲁迪乌斯的儿子呢""现在的你——就算想要保护——也无法完全保护好！"
- 认可儿子的觉悟后亲自传授「光之太刀」

### 六面世界3篇：红妈妈的期许
- 对齐格无职期间最常皱眉——看不惯孩子停滞不前
- 齐格告别时最先开口："那个友人是你想保护的人吧？"
- 露出久违的笑容并说"下定决心了吧——加油"
- 在齐格决定"保护帕库斯"时——他终于符合了红妈妈一直以来的期待

## 能力演变

### 剑术
- **第01~02卷**：剑神流中级→上级（两年内），开始产生"斗气"
- **第05卷**：剑神流上级技「无音之太刀」、「光之太刀」；战斗节奏不合常规、没有预备动作的斩击。被瑞杰路德评价"面对一般C级冒险者可空手压制"
- **第09卷**：获封剑圣——剑神直属弟子
- **第10卷**：接受北帝奥贝尔的训练（学习北神流及距离控制）
- **第13卷**：与水王伊佐露缇交手后学习抑制杀气；利用瑞杰路德传授的诱导技术战胜妮娜；三方僵局：艾莉丝→妮娜→伊佐露缇→艾莉丝
- **第14卷**：获封剑王（免许皆传）——将"光之太刀"改造为牺牲威力换取速度
- **第15卷**：掌握剑神流第三阶段——二刀流+北神流掷剑技+"光之太刀"；正式对决中几乎斩断奥尔斯帝德手腕
- **第17卷**：独自打倒两名北王（双胞胎纳库尔贾德）；真剑对决中正式超越基列奴
- **第25卷**：亲手斩杀前剑神加尔·法利昂
- **剑术风格总结**：从纯粹的剑神流→融合北神流距离控制和魔族诱导技术→独树一帜的"狂剑"

### 学识与教养
- **第02卷前**：不会读写、不会算术、不懂礼仪
- **第02卷后**：读写几乎完美、学会除法、能模仿淑女举止
- **语言**：人类语、魔神语（流利可吵架）、少量兽神语

### 魔术
- 学会初级魔术（第02卷），但无法做到无咏唱，中级也无望
- 天冷时能用魔术点火——仅实用级别

### 其他
- **烹饪**：完全不会（砍断砧板、烧成木炭）
- **育儿**：不擅长照顾小孩，但抱着婴儿哺乳时展现了母亲的一面
- **驯兽**：雷欧怕她——因为会被全力抱紧
- **教学**：教露西练剑、收神子为弟子、传授亚尔斯光之太刀

## 关键人际关系演变

### 鲁迪乌斯·格雷拉特（家庭教师 → 初恋 → 丈夫）
- **第02卷（7~10岁）**：从敌对（拳打脚踢）→ 信任（绑架事件被救）→ 喜欢（十岁承诺"五年后"）
- **第03~06卷（10~15岁）**：魔大陆相依为命 → 第06卷末初夜与离别
- **第07~14卷**：在剑之圣地修行——全部努力都是为了配得上鲁迪
- **第15卷（18岁）**：重逢 → 决斗 → 结婚——"算我赢了，可以吧？"
- **第16卷以后**：模拟战十战九胜，以剑守护鲁迪的背后
- **第26卷**：共生亚尔斯与克莉丝汀娜——先于鲁迪去世

### 基列奴·泰德路迪亚（师傅）
- **第02卷**：基列奴安排教艾莉丝剑术——"让她将来能当冒险者维生"
- **第04卷**：艾莉丝为基列奴辩护——"基列奴是我的师父！是我最尊敬的人！"
- **第15卷**：与基列奴一同骑马从剑之圣地出发赶往夏利亚
- **第17卷**：诀别的训练中真剑对决——正式超越基列奴，基列奴离去后放声大哭

### 瑞杰路德·斯佩路迪亚（战斗指导者）
- **第03~06卷**：魔大陆旅途中传授战斗技巧、探索动静等技能
- **第15卷**：在奥尔斯帝德战中配合剑神的教导
- **第25卷**：与瑞杰路德再次并肩作战——"在关键时机配合无间"

### 希露菲叶特（共妻 → 姐妹般的关系）
- **第15卷**：希露菲接纳艾莉丝——开导"你拥有我们没有的力量"
- **第16卷**：参加女子会、被希露菲安排时间
- **第17卷**：像对待不成材的妹妹般照顾
- 希露菲分娩时辅助找医生——与洛琪希和艾莉丝形成"前卫/后卫/治疗"的互补关系

### 洛琪希·米格路德（共妻 → 战友与朋友）
- **第22卷**：战斗中配合——用魔神语聊天加深友谊
- **第25卷**：被艾莉丝的坚定振奋——性格互补

### 子女
- **亚尔斯**：长子（红发），后来传授"光之太刀"
- **克莉丝汀娜**：次女——大嗓门和行动力遗传自艾莉丝
- 对露西：教她练剑但洗头太粗鲁被嫌弃

### 加尔·法利昂（剑神，修行对象）
- **第09卷**：剑神欣赏她的眼神和气魄，即日授予剑圣称号
- **第10卷**：剑神的哲学——"老子已经不会再拔掉徒弟的利牙了"
- **第14卷**：获封剑王称号后离开
- **第25卷**：亲手斩杀加尔——"成长了啊"——斩杀后发出悲痛呼号

### 妮娜·法利昂（竞争对手）
- **第09卷**：初到剑之圣地时击败妮娜——妮娜对她有屈辱→嫉妒→认同的复杂感情
- **第13卷**：三方僵局中形成奇妙关系——艾莉丝主动解释战术原理
- **第23卷**：被艾莉丝称为"那家伙"

## 重要经历时间线

- **第02卷（约9~12岁）**：初登场——无法驯服的野兽大小姐 → 绑架事件后被鲁迪救下开始改变 → 读写/算术/礼仪学习 → 十岁生日承诺"五年后" → 菲托亚领地消失事件（与鲁迪被转移至魔大陆）
- **第03卷（12~13岁）**：魔大陆冒险者——与鲁迪、瑞杰路德组建"Dead End"担任前卫 → 冒险者等级与队伍同步升至A级 → 学习魔神语
- **第04卷（13~14岁）**：晕船期间完全依赖鲁迪 → 走私事件中护卫孩子们 → 在德路迪亚村与兽族女孩建立友谊 → 离别时流泪
- **第05卷（约12岁）**：哥布林讨伐单独任务（首次实战杀人）→ 救助特蕾兹和神子 → 渡海前往中央大陆
- **第06卷（15岁）**：被瑞杰路德正式认定为"战士" → 奥尔斯帝德战中鲁迪被杀崩溃 → **与鲁迪初夜** → **不告而别前往剑之圣地**（剪掉红色长发）
- **第09卷（17岁）**：初到剑之圣地——"我对你这样的杂碎没兴趣！" → 即日获封剑圣 → 两年修行
- **第10卷**：空挥修行（半年）→ 与北帝奥贝尔遭遇战败 → 接受北帝训练学习北神流
- **第13卷**：与水王伊佐露缇交手学习对付水神流 → 三方僵局共同训练 → 一年后略占上风
- **第14卷**：击败妮娜·法利昂 → **获封剑王（免许皆传）** → 决定"想回家了"——与鲁迪写信时刻不谋而合
- **第15卷**：收到鲁迪信后立即出发 → 在鲁迪家门口犹豫一小时 → **奥尔斯帝德战救场** → 决斗后结婚——"算我赢了，可以吧？"
- **第16卷**：击退路克 → 参加女子会 → 模拟战碾压鲁迪 → 图书迷宫探险殿后
- **第17卷**：赤龙胡须之战斩杀大量敌兵 → **独自打倒两名北王** → 真剑对决中**正式超越基列奴** → 基列奴离去后放声大哭
- **第18卷**：孕期 —— 被日记提及怀有第三子（亚尔斯）
- **第19卷**：**产下长子亚尔斯**（红发）→ 鲁迪归来时母子平安
- **第20卷**：留守夏利亚——哺乳中的母亲形象、与妮娜一同游玩
- **第21卷**：被爱夏叫来米里斯保护鲁迪 → 收神子为弟子——"就能变得像我一样"
- **第22卷**：涅克罗斯要塞战——光之太刀砍入阿托菲脖颈三分之一 → 被毛茸茸使魔亲近陷入幸福恍惚
- **第23卷**：天大陆之旅——以光之太刀劈开霸王腭龙头颅 → 剑之圣地"狂剑王"风评
- **第24卷**：与洛琪希调查鬼岛 → 想吃冰魔法保存食物 → 赶往斯佩路德村——"我想快点去见瑞杰路德"
- **第25卷**：作为"Dead End"成员重组 → **亲手斩杀前剑神加尔·法利昂** → 战后毕黑利尔国王护卫
- **第26卷**：总力战对抗斗神——被重创 → 被希露菲救回 → 十年后生下克莉丝汀娜 → **70岁+在锻炼后去世** → 鲁迪哭了三天

### 蛇足篇
- 诺伦婚礼 → 与儿子亚尔斯死斗 → 认可后传授光之太刀 → 克莉丝毕业典礼

### 六面世界3篇
- 对齐格无职期间最常皱眉 → 齐格告别时最先开口肯定
