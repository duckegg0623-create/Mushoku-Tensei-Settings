# 洛琪希·米格路德 / 洛琪希·米格路迪亚·格雷拉特（Roxy Migurdia / ロキシー・ミグルディア）

## 基本信息
- **全名**：洛琪希·米格路迪亚·格雷拉特（Roxy Migurdia Greyrat）
- **通称**：洛琪希（Roxy）、蓝妈妈（Blue Mama，子女称呼）、师傅（鲁迪专称）
- **性别**：女
- **种族**：魔族·米格路德族（魔大陆比耶寇亚地方出身）
- **年龄**：初登场约40余岁（米格路德族寿命约200岁，外表约10岁时即成成人形态后几乎不变）
- **出生地**：魔大陆米格路德村
- **父亲**：洛因（Loxin）
- **母亲**：洛嘉莉（Rokari）
- **配偶**：鲁迪乌斯·格雷拉特（第二夫人）
- **子女**：菈菈·格雷拉特（长女）、莉莉·格雷拉特（次女）
- **职业**：魔术师 → 家庭教师 → 冒险者 → 西隆王国王子家庭教师 → 拉诺亚魔法大学教师 → 魔法大学校长
- **称号**：水圣级魔术师 → 水王级魔术师
- **所属**：原无固定所属，后为格雷拉特家成员、奥尔斯帝德阵营协力者

## 外貌特征演变

### 初登场（第01卷）
- 水蓝色头发绑麻花辫（在特定光线下显绿，容易被误会为危险种族）
- 雪白肌肤，微眯的眼眸，娇小端正
- 娇小体型（是她的自卑点，"我才不小"）
- 冷淡的面部表情，略有困意的眼睛
- 常穿魔术师风格的褐色长袍，随身携带魔杖

### 中期（第02~13卷）
- 外表几乎无变化（米格路德族特性：约10岁即成人形态后不再变化）
- 在不同场合穿相应服装：冒险者装束、教师装束、日常家庭便服
- 被鲁迪评价为"外观如国中生"

### 后期（第14~26卷）
- 怀孕期间肚子明显隆起，产后恢复迅速
- 作为母亲和妻子的气质逐渐显现
- 直至鲁迪晚年（74岁），外表依然几乎不变——被说"没变"会生气（这是夸奖）

## 性格与心境演变

### 第01卷：青年教师的自信与自卑
- 表面冷淡，但内心容易动摇
- 经不起夸奖（被称赞时会得意地嘴角上扬）
- 认真、努力，每晚为教学预习到很晚
- 有点笨手笨脚，经常犯小错
- 守财奴倾向（帮村民用魔术收钱）
- 不擅应对被尊称"师傅"（自觉会被弟子超越）
- 处于青春期的好奇和欲望
- 身为魔族受到偏见歧视（曾被侮辱为"肮脏的低贱魔族"、"乡巴佬"）
- 与自己的师傅关系不佳

### 第02卷：独立的宫廷教师
- 对自己的教师才能缺乏自信（将帕库斯和鲁迪乌斯对比后觉得无法教导普通学生）
- 被帕库斯性骚扰时冷静反击，辞职时击退追随的骑士
- 对米格路德族的身材感到自卑
- 为鲁迪编写魔神语教科书（不到一年完成），对米格路德族项目写了五页最详细
- "遇过鲁迪乌斯那种学生后，或许再也无法担任教师"

### 第03~04卷：冒险者与搜索者
- 冒险者时期曾有三名新手队友（"利卡里斯愚连队"），花了五年升上A级（而非诗歌声称的两年）
- 艾莉娜丽洁的滥交行为让她精神崩溃——对恋爱仍怀有理想
- 在魔大陆搜索鲁迪，效率至上主义（也因此错过与鲁迪相遇）
- 过去拼命想被当成大人（吹嘘自己无所不能），现在因徒弟过于优秀而既自豪又惭愧

### 第05~06卷：归乡与自我确认
- 回到米格路德村——感到疏离和寂寞（无法使用心电感应能力）
- 通过拥抱与父母和解——"话语并不能代表一切"
- 整合情报终于承认"Dead End的饲主"就是鲁迪，但因错过见面而"无脸见他"
- 从奇希莉卡处获取格雷拉特全家下落，决定返回米里希昂报告
- 艾莉娜丽洁的忠告："一旦碰上真心喜欢的对象要主动出击"
- 对艾莉娜丽洁可能"对鲁迪出手"感到不愉快

### 第07~11卷（间接行动期）
- 带着情报返回米里希昂向保罗传达
- 见到瑞杰路德时极度恐惧（斯佩路德族是米格路德族童年恐怖故事主题）
- 对鲁迪的感情在不知不觉中已超越单纯的师徒之情

### 第12卷：命运的转折——从师傅到妻子
- 在转移迷宫中独自被困一个月，精神与肉体濒临极限
- **核心事件**：被鲁迪救出时没认出长大后的鲁迪，笑着说"初次见面"——鲁迪崩溃呕血
- 对此极度羞愧，此后耿耿于怀
- 保罗死后，决心以身体安慰崩溃的鲁迪，深夜造访发生关系
- 以简单直率的温柔治愈了崩溃的鲁迪——听完"虚构故事"，给出"珍惜现在的家人就好"的建议
- 被鲁迪请求娶为第二夫人——诺伦激烈反对时，洛琪希差点主动退出离开
- 最终被希露菲和家族接纳

### 第13~15卷：成熟的第二夫人
- 传授鲁迪水王级魔术"雷光"——作为师傅的余晖
- 与希露菲建立良好平衡，双方都有建立良好关系的意志
- 发挥"可靠的大姐"角色——在家庭中起着调和与推动的作用
- 察觉鲁迪不对劲（人神梦），主动保护家庭
- **宣布怀孕**（约三个月）——在全家齐聚时正式宣布
- 协助魔导铠制作的全体监督
- 对鲁迪的隐瞒直言："以前的鲁迪明明是不会撒谎的好孩子"
- 促进艾莉丝融入——直接问鲁迪"何时要举办宴会庆祝和艾莉丝结婚？"

### 第16~18卷：母亲角色
- 怀孕中——要求鲁迪在出发前取好名字（男孩"罗洛"、女孩"菈菈"）
- 临别嘱托："请你要平安回来。我想要和鲁迪一起抱这个孩子"
- 战术指导：指出近距离对抗剑士的正确方式——"向前闪过攻击，互换彼此位置"
- 女儿菈菈被圣兽雷欧认定为"救世主"——暗示洛琪希的命运重要性
- 人神曾积极阻止鲁迪与洛琪希结合——说明洛琪希是关键的命运节点

### 第19~22卷：战士与母亲的双重身份
- 坚持陪同鲁迪前往西隆（菈菈大哭仍不改变决定）
- 卡隆堡垒之战：在屋顶绘制火圣级魔法阵
- 与死神的战斗中濒临死亡——防护装备全部损毁
- 以身作则安慰害怕杀人的鲁迪——"杀人的感觉不会消失但可以习惯"
- 涅克罗斯要塞之战：与艾莉丝配合击败土之培立托德（泥沼妙招）
- 回米格路德村与鲁迪正式完成婚礼——带着菈菈与聘礼
- 与艾莉丝用魔神语聊天加深友谊
- 开导希露菲："差别待遇这种事，我累积了不少经验！"

### 第23~25卷：可靠的支柱
- 开导产后情绪低落的希露菲——点出真正的困扰"更在意头发是绿色的"
- 在天大陆之旅中配合希露菲共同以魔术攻击
- 毕黑利尔王国任务中与艾莉丝搭档——冷静分析、小心谨慎
- 相信"家里有希露菲在不会有事的"

### 第26卷：王牌运送者与晚年
- **核心贡献**：将零式从夏利亚挖出、画好转移魔法阵、运至战场——没有她鲁迪将战败
- 因先挖零式后画转移魔法阵的顺序搞错——花了大量时间，累得魔力枯竭兼熬夜
- 十年后（34岁）：外表完全没变，仍作为老师指导鲁迪——"人生就是会有许多失败"
- 生下三女莉莉
- 鲁迪临终时最后进入房间——流泪说"卢迪——真是辛苦你了"
- 鲁迪叫她"洛克希老师"——全卷最感人时刻
- 历史地位：洛克希·M·格瑞拉特——魔法大学校长——400年代代表性魔法师
- 将鲁迪的无咏唱魔法方式写成论文确立学习方法——被魔法三大国与阿斯拉王国采纳
- 知名度甚至高于鲁迪本人

## 能力演变

### 魔术
- **水系统**：圣级（第01卷）→ 王级（第02卷，信中告知已学会）
- **攻击魔术**：各系统中级-上级
- **治愈魔术**：中级
- **火圣级魔法阵**：第19卷在卡隆堡垒屋顶绘制
- **混合魔术「泥沼」**：第22卷与艾莉丝配合击败培立托德
- **无咏唱**：不会无咏唱（认为这是极其罕见的高级技巧），但学会了缩短咏唱
- **雷光**：水王级魔术，传授给鲁迪
- **不会召唤魔术**

### 教学能力
- 擅长因材施教，能配合学生理解程度调整课程
- 教学易懂有趣，互动式授课
- 教的不仅是魔术，还包括杂学、世界观等
- 第26卷：将无咏唱魔法方式写成论文，确立学习方法，被各国采纳

### 其他能力
- **语言**：人类语、魔神语
- **冒险者经验**：A级冒险者，花了五年升上A级
- **料理**：简单料理难不倒（以前的冒险者经验）
- **魔法阵**：对魔法阵多多少少有研究

### 弱点与局限
- 不会无咏唱魔术
- 不会召唤魔术
- 不能使用米格路德族的心电感应能力（只能听到类似杂音的声响）
- 治愈魔术仅中级
- 身材娇小（自卑来源）

## 关键人际关系演变

### 鲁迪乌斯·格雷拉特（弟子 → 丈夫）
- **第01卷（鲁迪3~5岁）**：家庭教师，倾囊相授两年。鲁迪是她第一位（可能也是唯一）超越她的弟子。带领鲁迪首次踏出家门，治愈了他的外出恐惧症。赠予自制魔杖和米格路德族护身符。离开后写信言语中带着"弟子超越师傅"的复杂心情
- **第02~06卷**：为鲁迪编写魔神语教科书。收到鲁迪信件时会开心（感觉信件似乎有香味）。在魔大陆搜索鲁迪行踪
- **第12卷**：迷宫重逢（没认出长大后的鲁迪）→ 以身体安慰崩溃的鲁迪 → 成为第二夫人
- **第13卷**：传授水王级魔术——作为师傅的集大成
- **第15卷**：宣布怀孕——"请你让我幸福"
- **第22卷**：回故乡正式完成婚礼
- **第26卷**：临终时鲁迪叫她"洛克希老师"

### 希露菲叶特（共妻 → 亲友）
- **第13卷**：希露菲坦言曾嫉妒洛琪希——"鲁迪提起洛琪希时总是充满崇敬"
- **第15卷**：洛琪希发挥大姐风范，协调家庭关系
- **第23卷**：以自身经历开导希露菲走出产后心理创伤——点出"差别待遇"的核心问题

### 艾莉丝（共妻 → 战友兼亲友）
- **第15卷**：接纳艾莉丝为第三夫人
- **第22卷**：战斗中配合默契——用魔神语聊私密话题，友谊加深

### 双亲（洛因与洛嘉莉）
- **第03卷**：回到米格路德村，父母对女儿提到的"鲁迪乌斯"印象深刻
- **第05卷**：因无法使用心电感应感到疏离，但通过拥抱明白彼此的爱——"话语并不能代表一切"
- **第22卷**：带鲁迪和菈菈回故乡正式完成婚礼

### 艾莉娜丽洁·杜拉冈罗德
- **第04卷**：在魔大陆组队搜索，对其滥交行为无法理解和接受
- **第06卷**：接受艾莉娜丽洁的恋爱建议——"一旦碰上真心喜欢的对象要主动出击"
- **第12卷**：受艾莉娜丽洁暗示，以身体安慰鲁迪

### 鲁迪的子女们
- **菈菈**：亲生女儿，被称为"救世主"，圣兽雷欧对其特别忠诚
- **莉莉**：次女（十年后出生）
- **露西、亚尔斯、齐格、克莉丝汀娜**：继子女，平等对待

## 重要经历时间线

- **第01卷**：从拉诺亚魔法大学毕业 → 成为鲁迪的家庭教师（两年间）→ 离开布耶纳村 → 以冒险者身份进入迷宫打出名号 → 被西隆王国聘为王子家庭教师 → 晋升水王级
- **第02卷**：担任帕库斯·西隆的家庭教师 → 被帕库斯性骚扰 → 冷静反击后辞职 → 为鲁迪编写魔神语教科书（约半年内完成）→ 注意到菲托亚领地异变 → 前往菲托亚领地搜索 → 发现鲁迪等人失踪 → 决定前往魔大陆搜索
- **第03卷**：抵达魔大陆 → 回到米格路德村与父母重逢 → 与鲁迪擦肩而过（鲁迪之后抵达同村）
- **第04卷**：抵达温恩港 → 与艾莉娜丽洁和塔尔韩德组队 → 在魔大陆各城市搜索 → 与鲁迪再次擦肩
- **第05卷**：回米格路德村归乡（心灵疏离→拥抱和解）→ 在利卡里斯镇与旧队友诺克巴拉重逢 → 确认"Dead End的饲主"就是鲁迪 → 决定不去见面继续搜索（无脸见面）
- **第06卷**：在克拉斯玛镇与奇希莉卡同桌共饮 → 获取格雷拉特全家下落情报 → 兵分两路：返回米里希昂向保罗报告
- **第07~11卷**：返回米里希昂传递情报 → 在东部港护卫诺伦等人 → 直面童年恐惧（斯佩路德族）
- **第12卷**：在转移迷宫中被困一个月濒临极限 → 被鲁迪救出 → **没认出长大后的鲁迪** → 保罗死后以身体安慰鲁迪 → 成为第二夫人 → 归程回家
- **第13卷**：传授鲁迪水王级魔术"雷光" → 与鲁迪和希露菲实现三人约会/同寝 → 戴上鲁迪赠予的结婚贺礼帽子
- **第14卷**：被挡在空中要塞门外（魔族不得入内）→ 察觉鲁迪样子奇怪 → 陪伴鲁迪度过未来自己到访的事件
- **第15卷**：察觉鲁迪异常，冷静分析"比起自己，家人遭逢不幸更让鲁迪难过" → 协助魔导铠制作 → **宣布怀孕**（约三个月）→ 促进艾莉丝融入 → 在场见证鲁迪与奥尔斯帝德一战
- **第16卷**：怀孕中——要鲁迪出发前取名（"罗洛"或"菈菈"）→ 战术指导鲁迪对抗剑士 → 雷欧对她特别忠诚
- **第17卷**：鲁迪不在时身体发生变化——在意隆起的肚子和凸出的肚脐 → 鲁迪归来后要求他舔肚子 → 感到胎动——"应该是在向爸爸说欢迎回家吧"
- **第18卷**：生下菈菈——菈菈被雷欧认定为"救世主"
- **第19卷**：坚持陪同鲁迪前往西隆 → 卡隆堡垒绘制火圣级魔法阵 → 与死神战斗濒临死亡 → 向鲁迪坦白曾杀过人
- **第20卷**：留守夏利亚兼教职 → 出席毕业典礼，受女学生们感谢
- **第21卷**：留守夏利亚（未直接出场）——被鲁迪委托在拉诺亚提出通缉基斯申请
- **第22卷**：参与阿托菲战（泥沼妙招与艾莉丝配合）→ 与艾莉丝用魔神语聊天加深友谊 → **与鲁迪回故乡正式结婚**
- **第23卷**：开导产后情绪低落的希露菲——"差别待遇这种事，我累积了不少经验！" → 天大陆之旅 → 以"冰霜击"攻击霸王腭龙翅膀
- **第24卷**：与艾莉丝在第三都市调查鬼岛 → 发现转移魔法阵停摆 → 冷静分析原因 → 赶往斯佩路德村
- **第25卷**：与艾莉丝通讯中断后前往斯佩路德村（侧面描写较多）
- **第26卷**：**将零式运至战场**（关键贡献）→ 十年后生下莉莉 → 晚年任魔法大学校长 → 鲁迪临终时被叫"洛克希老师" → 历史评价：洛克希·M·格瑞拉特——400年代代表性魔法师

### 六面世界3篇（40岁+）
- 为齐格张罗与莎莉爱尔相亲（欲强化与阿斯拉王国关系）→ 齐格拒绝后大度接受 → 教育方针："随心所欲的活着，有不懂的东西就问问别人"
