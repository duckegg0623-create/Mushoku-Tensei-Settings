# 札诺巴·西隆（Zanoba Shirone）

## 基本信息
- **全名**：札诺巴·西隆（Zanoba Shirone）
- **通称**：札诺巴、拔头王子（Head-Chopping Prince）
- **性别**：男
- **种族**：神子（拥有特殊能力的人类——怪力，以肌肤感觉迟钝为代价）
- **身份**：西隆王国第三王子 → 被流放者 → 魔法大学特别生 → 人偶研究家 → 西隆王国卡隆堡垒指挥官 → 平民（放弃王族身份） → 札诺巴商行人偶部门负责人 → 奥尔斯帝德阵营成员 → 人偶商会会长
- **配偶**：无（不打算结婚——因肌肤感觉迟钝无法与活人建立亲密关系）
- **弟子/共同生活者**：茱丽叶特（茱丽）
- **师傅**：鲁迪乌斯·格雷拉特（人偶制作之师）
- **称号**：拔头王子、神子、怪力的神子
- **所属**：原六魔连之一、奥尔斯帝德阵营、札诺巴商行

## 外貌特征演变
- **第06卷（初登场）**：西隆王国第三王子——王族气质但眼中只有人偶
- **第08-10卷（魔法大学时期）**：普通青年学生外表，无特殊变化
- **第16卷**：通宵制作护手后的憔悴——但对人偶的热情不减
- **第20卷（约22岁）**：在夏利亚盖房定居——开始平民生活
- **第26卷**：战斗中被扯烂衣服，穿着斯佩路德族民族服饰继续战斗

## 性格与心境演变

### 第06卷（初登场）：人偶狂热的怪力王子
- **背景**：神子之力导致"手会滑掉"杀了婴儿弟弟和新娘——被当成残忍的"拔头王子"
- 实际缘由是神子之力无法控制——非本意
- 人偶（尤其是洛琪希人偶）的狂热收藏家
- 一眼认出鲁迪就是人偶制作者——当场磕头拜师
- 展现神子之力轻松抓住帕库斯头将其控制——迫使第一第二王子介入调停
- 名义上被流放至国外——实为将危险人物逐出权力中心

### 第08-09卷（魔法大学时期）：追随师傅与人偶之路
- 对师傅鲁迪绝对忠诚——在特别生教室重逢时兴奋冲撞高呼"师傅！"
- 对女性冷淡（尤其是活人）——曾裸体抱铜像被撞见
- 有幽默感和自嘲能力——坦然提及自己杀死的弟弟
- 无法使用无咏唱魔术（魔力总量不足），手工制作因怪力导致手指不灵活
- **购买茱丽**：在菲兹建议下购买矿坑族少女为奴隶代为制作——将茱丽视为师妹而非奴隶
- 洛琪希人偶被莉妮亚和普露塞娜践踏破坏——仍将碎片一一捡回保管。看到修复后的人偶立即原谅两人
- 与巴迪冈迪讨论人偶必要条件——讨好巴迪以获得制作奇希莉卡人偶的许可

### 第10-11卷（忠诚与暴力）：主从关系的考验
- 第10卷：协助七星研究——提出"多重构造魔法阵"解决思路（来自自动人偶研究）。七星崩溃时以公主抱将其送至医务室
- 第11卷（闲话）：金洁私下批评鲁迪演讲"无聊"→札诺巴愤怒扼其脖子——鲁迪斥责他"不可以因为这点小事就杀部下"
- **学习价值观**：从鲁迪处学到"不能因为小事杀部下"——一个不会因被侮辱就杀人的人

### 第13卷（约17岁）：札里夫义手——技术天赋的释放
- 在鲁迪旅行期间（半年）完成义手试作品——以两人名字命名为"札里夫义手"
- 可按照使用者意志动作，甚至能抑制札诺巴自身的怪力诅咒
- 切开人偶躯体发现核心——由多个表面刻满魔法阵的小魔石组合而成。"仿佛这件事正是他的使命"
- 洛琪希行五体投地大礼——"她是师傅的师傅"

### 第14-15卷（18岁）：战斗中的力量与未来悲剧的启示
- 第14卷：随鲁迪前往空中要塞——对佩尔基乌斯城堡中的艺术品有浓厚兴趣，与其展开艺术对话。魔大陆之战——抓住阿托菲手臂→她自断手臂脱离→札诺巴一拳将其打穿墙壁
- 第15卷：负责魔导铠整体设计及驱动部分——兴奋接受"魔导铠"概念
- **未来日记中的札诺巴**：与鲁迪一起制作自动人偶→被神殿骑士团杀害——"在地下室门口被烧成焦炭"。鲁迪为此悲痛万分

### 第16卷（约18-19岁）：对师傅的忠诚超越生命
- 夜间独自去回收被破坏的魔导铠——撞见奥尔斯帝德→被打倒→即使倒下仍爬向铠甲"绝对不会把这副铠甲交出去，一定得带回去给你"
- 对奥尔斯帝德的恐惧受诅咒强烈影响——"那家伙根本就是恶魔""居然会为了那种人卖命，师傅您肯定是疯了"
- 发誓："在下次战斗之前，本王子会制作一套连茱丽都能装备的铠甲给您看"

### 第18-19卷（约20岁）：西隆政变——从王族到平民
- 第18卷：茱丽制作超越札诺巴的杰作→他感动落泪。西隆政变——帕库斯杀害父王与王兄夺取王位→札诺巴收到归还敕命
- 第19卷：明知是陷阱仍接受——"国要臣死，臣不得不死"。被任命为卡隆堡垒指挥官——仅五百兵力对抗五千敌军（十倍差距）
- **奇袭作战**：亲率百人突袭敌阵擒获敌国王族——仅十人生还、九十人战死
- **帕库斯之死**：得知杰伊德将军叛乱→赶往救援帕库斯——但仍未能赶上→帕库斯跳楼自尽——"帕库斯是我的弟弟——最后一个留在世上的血亲"
- **最大转变**：放弃王族身份——"我要成为平凡的札诺巴"。加入奥尔斯帝德阵营——"人神的目的就是让你无法集结伙伴——那我帮你集结就好"

### 第20卷（约22岁）：新生活——平民札诺巴
- 变卖王族物品——在鲁迪家附近盖房（二层建筑：一楼工房、二楼居住空间）
- 与金洁、茱丽三人共同生活
- 被鲁迪任命为人偶贩卖部门负责人——被鲁迪和约瑟夫视为"兄弟"
- 札诺巴在模拟战中对布鲁克林输多赢少（纯魔法战斗不利）

### 第24-26卷（约23-34岁）：忠诚与战斗
- 第24卷：在毕黑利尔首都搜集情报→转移魔法阵停摆后急忙前往斯佩路德村。判断第二都市伊雷尔是敌人本营——吹响佩尔基乌斯笛子无反应
- 第26卷：参与斗神总力战——与冬迦担任前排防御。双手接住斗神双拳→被第三拳击中腹部飞出。被希露菲救回
- 历史评价：阿苏拉史料记载为原西隆王国王子——人偶商会会长——怪力的神子

### 蛇足篇：自动人偶开发成功
- 与鲁迪耗费接近十年开发完全自律人偶——经历无数次失败终于成功启动
- 试作4号机为近乎完成品——外观与七星极为相似。札诺巴为人偶的"父亲"——称呼为"我的爱女"
- 不打算结婚——肌肤感觉迟钝，"活生生的女人肌肤太过柔软得不到足够刺激"
- 鲁迪建议与茱丽结婚→札诺巴婉拒（"无法做出子女的对象做伴侣——对方会很辛苦"）
- 茱丽对札诺巴买色情人偶会不高兴——会把人偶收起来

## 能力演变

### 神子能力
- **怪力**：可徒手扭断人首、抓住阿托菲手臂使其自断脱离、一拳打穿墙壁、双手接住斗神双拳
- **代价**：肌肤感觉迟钝——触感弱→无法拥有正常亲密关系
- **速度劣势**：速度慢、跑步不行——"练跑不够"

### 魔术与魔法技术
| 时期 | 能力 | 备注 |
|------|------|------|
| 第08卷 | 土魔术（初级） | 刚学习，魔力总量不足 |
| 第10卷 | 多重构造魔法阵理论 | 从人偶研究获得灵感 |
| 第13卷 | 札里夫义手 | 与克里夫共同开发——可按意志动作 |
| 第15卷 | 魔导铠整体设计/驱动 | 负责物理设计和驱动部分 |
| 第20卷 | 模拟战中输多赢少 | 纯魔法战斗不利 |

### 制作与工艺
- **人偶制作**：极度热衷但精细操作困难（怪力导致）
- **魔导铠开发**：一式→一式改→二式→二式改→零式——核心贡献者（驱动设计、物理构造）
- **札里夫义手**：创造性设计与实用性结合
- **自动人偶开发**：与鲁迪十年开发成功——试作1号至4号机
- **研发副产品**：蛙袋避孕套（尤其畅销）

## 关键人际关系演变

### 鲁迪乌斯·格雷拉特（师傅/挚友/"兄弟"）

- 第08-10卷：追随师傅学习人偶制作——绝对忠诚
- 第11卷：被鲁迪斥责不能因小事杀部下——学习新价值观
- 第13-16卷：通宵合作开发札里夫护手和魔导铠
- 第16卷：即使面对奥尔斯帝德也誓死保护师傅的铠甲
- 第19卷：称鲁迪为"兄弟"——"礼尚往来"帮助鲁迪创立公司
- 蛇足篇：共同耗费十年开发自动人偶
- **关系本质**：从师徒→挚友→兄弟。未来日记中始终陪伴颓废的鲁迪直至被害

### 茱丽叶特（弟子/共同生活者）
- 第08卷：购买矿坑族少女代为制作人偶——命名为茱丽叶特。视为师妹而非奴隶——悉心照顾

- 第18卷：茱丽制作超越札诺巴的杰作——他感动落泪。关系已超越主仆

### 金洁·约克（忠诚亲卫队）

- **前史**：西隆王国第三王子→神子之力失控杀死婴儿弟弟和新娘→被称为"拔头王子"

- **第09-10卷**：继续人偶制作→讨好好巴迪冈迪→协助七星魔法阵研究→七星崩溃时救助
