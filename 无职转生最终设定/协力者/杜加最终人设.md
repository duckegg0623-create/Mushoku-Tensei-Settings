# 杜加·伽鲁多（Doga Galdo）

## 基本信息
- **全名**：杜加·伽鲁多（Doga Galdo），通称「冬迦」
- **种族**：鬼族（食人魔）——也可能是斯佩路德族（来源有矛盾，第24卷称斯佩路德族，第25/26卷称鬼族）
- **性别**：男
- **身份**：阿托菲拉托菲的忠诚部下 → 鬼族战士 → 阿斯拉七骑士之一「王的门卫」
- **所属**：魔大陆阿托菲阵营 → 阿斯拉王国黄金骑士团
- **外观**：大块头、木讷寡言、存在感薄弱、头盔内部的脸和气地笑着
- **战力评**：阿斯拉王国前五强实力者

## 外貌特征演变
- **第22卷**：阿斯拉王城门卫——初步接触文明社会
- **第24卷**：大块头，木讷寡言，存在感薄弱；被认为「没有骗人的脑袋」
- **第25卷**：鬼族战士——在谷底与地龙搏斗
- **第26卷**：参与对抗斗神总力战
- **蛇足篇**：身着黄金铠甲，手持巨大战斧，表情和蔼，笑容完全没有邪念——极其淳朴

## 性格与心境演变

### 第22-24卷——守护者的萌芽
- 第22卷：在阿斯拉王城任门卫
- 第24卷：随鲁迪前往首都毕黑利尔，作为护卫；抱着女性患者慌张奔走（疫病事件）；被认为还不太派得上用场——「有总比没有好」
- 回程因睡过头没跟来——鲁迪在桥上遇袭时若杜加在场可能改变战局

### 第25卷——最关键的一次救援（核心）
- **最忠诚的行动**：从阿托菲处受领黑色左臂——命「去救鲁迪乌斯」
- 跳入地龙谷——以自己的身体为缓冲接住坠落的鲁迪
- 在谷底河滩发现鲁迪——克服大量地龙的威胁将其救出
- 将阿托菲的黑手臂移植给鲁迪——手臂完美融合
- 在鲁迪身体不便期间担任保护者——击退地龙
- 与鲁迪一同探索龙族遗迹——发现蓝色液体
- 搬运无法行动的鲁迪——利用遗迹知识和蓝色液体逃出地龙谷
- **关键意义**：杜加是鲁迪坠谷不死的关键人物——没有他鲁迪早已死亡

### 第26卷——前排防御者
- 总力战中与扎诺巴担任前排防御——挡下斗神攻击
- 两人相互掩护——避免对方受到直击——但仍会骨折或受伤
- 克里夫昏迷后失去治疗支援——最终被击飞
- 被希露菲的无咏唱治愈魔术救回——重新站起
- 听到翔德尔被打倒露出一脸哭相

### 蛇足篇——阿斯拉七骑士
- 成为阿斯拉七骑士之一「王的门卫」
- 身着黄金铠甲手持巨大战斧——站在爱丽儿私人房间前
- 对伊佐露缇：不收缴武器、不进行身体检查——从头发上摘下一片花瓣——「因为伊佐露缇很漂亮——不能粘上」
- 递水给出后的伊佐露缇——「请——已经冷了」
- 表情和蔼笑容完全没有邪念——极其淳朴
- 伊佐露缇判断：若杜加认真起来——自己无法突破他到达门的后面

## 能力演变

### 战斗/特殊能力
| 时期 | 能力 |
|------|------|
| 第22-24卷 | 作为护卫「有总比没有好」——尚未成熟 |
| 第25卷 | 独自在谷底对抗地龙群，保护鲁迪确保其生存 |
| 第26卷 | 前排防御者——与扎诺巴配合——虽不及鬼神但坚韧 |
| 蛇足篇 | 阿斯拉王国前五强实力者；伊佐露缇判断无法突破其防线 |

### 装备与持有物
- 巨大战斧（蛇足篇）
- 黄金铠甲（蛇足篇·阿斯拉七骑士装备）

## 关键人际关系演变

### 阿托菲拉托菲（主人）
- 绝对忠诚——毫不犹豫执行阿托菲的命令
- 第25卷受阿托菲之命跳谷救鲁迪
- 完全信赖阿托菲的判断

### 鲁迪乌斯·格雷拉特
- 第24卷：担任鲁迪护卫（效果有限）
- 第25卷：（核心）救命恩人——在谷底的共同生存经历建立了友谊
- 移植阿托菲左臂给鲁迪

### 伊佐露缇·克尔埃尔（微妙好感）
- 蛇足篇：对伊佐露缇表现细腻一面——摘花瓣、递冷水
- 不收缴武器也不身体检查——因为「伊佐露缇是爱丽儿大人的骑士」
- 被信赖的感觉让伊佐露缇心中产生鼓动——但仍说「这个脸不行吧」
- 伊佐露缇觉得「能毫不犹豫地把背后交付给他」

### 扎诺巴·西隆
- 第26卷共同担任前排防御——彼此掩护配合默契

## 重要经历时间线
- **第22卷**：在阿斯拉王城任门卫
- **第24卷**：随鲁迪前往毕黑利尔；回程睡过头未能陪同——间接导致鲁迪遭袭时缺援
- **第25卷**：（核心）奉命跳谷救鲁迪；移植阿托菲左臂；护送鲁迪穿越地龙谷
- **第26卷**：参与对抗斗神总力战——前排防御者
- **蛇足篇**：成为阿斯拉七骑士「王的门卫」；与伊佐露缇有微妙互动
