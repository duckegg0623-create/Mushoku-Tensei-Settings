# 妮娜·法利昂 / 妮娜·布里兹（Nina Farion / Nina Blitz）

## 基本信息
- **全名**：妮娜·法利昂（Nina Farion），婚后名「妮娜·布里兹（Nina Blitz）」
- **性别**：女
- **种族**：人族
- **身份**：剑神加尔·法利昂的长女 → 剑圣（16岁） → 剑王 → 剑神的妻子
- **所属**：剑之圣地 → 剑神流
- **特征**：略为浅黑的肌肤、深蓝色的头发、锐利的眼神
- **年龄**：第09卷时18岁 → 第24卷时怀孕

## 外貌特征演变
- **第09卷（18岁）**：剑圣——被评价为将来必成剑帝的天才
- **第23卷**：略为浅黑的肌肤、深蓝色头发、锐利的眼神
- **第24卷**：怀孕中——无法前来支援
- **蛇足篇**：成长为更成熟的剑士，成为吉诺的妻子和剑神

## 性格与心境演变

### 第09卷——嫉妒与成长
- 急性子（遗传自父亲），自尊心强
- 16岁获剑圣称号，被认为20岁会成为剑王，25岁前成剑帝
- 为剑而活，认为自己与恋爱无缘
- **初遇艾莉丝**：对艾莉丝的粗暴态度愤怒，被折断木刀后用拳头暴打面部——流鼻血、断齿、失禁
- 此后两年没和艾莉丝好好说话
- 排斥艾莉丝加入弟子圈子——逐渐意识到自己的「天真」
- 被艾莉丝「不是处女」的发言彻底击溃
- 委托情报贩子调查「鲁迪乌斯」是否真实存在——得知确有其人后决定去魔法大学打倒鲁迪
- **魔法大学之行**：亲历巴迪冈迪的全灭事件——自己的斩击对巴迪无效；目睹鲁迪一击轰飞魔王上半身——精神受巨大冲击后默默返回
- **转变**：回到剑之圣地后更加努力——携带两把剑、不再小看艾莉丝的拳击法、疏远浅交的同年代弟子
- 成为艾莉丝名符其实的对手

### 第20卷——社交圈扩大
- 收到伊佐露缇来信——前往阿斯拉参加爱丽儿加冕祭典
- 看不懂文字——请门生代为朗读——当机立断前往（「剑神流的座右铭」）
- 与艾莉丝、伊佐露缇三人到处游玩
- 在宴会中误以为鲁迪威胁爱丽儿试图砍他——实际是误会

### 第23卷——守护者姿态
- 在剑之圣地以为有人来踢馆——带真刀赶来
- 对剑神的称呼——纠正艾莉丝「至少要称呼他师傅」
- 释放杀气吓唬鲁迪——后说是「被现场气氛影响」
- 向其他门徒说明艾莉丝是「狂剑王」——名字相当有说服力

### 第24卷——怀孕
- 怀孕中——无法支援毕黑利尔王国
- 艾莉丝对她怀孕感到开心

### 蛇足篇——求婚与归宿
- **童年**：是剑之圣地所有道场孩子的老大，组织秘密组织持续5年；剑术最强，吉诺是唯一能讨论剑术理论的同龄人
- **成长**：输给艾莉丝后深深受辱，在无人处哭了
- 对艾莉丝进行人身攻击试图孤立——但艾莉丝根本不在乎——无效
- 突然独自离开圣地两个月后完全改变：不再攻击艾莉丝、对剑术更真挚、收起傲慢、专注特训、不再把吉诺当小弟
- **求婚**：参加阿斯拉戴冠仪式回来后——对吉诺说「呐吉诺——我们也结婚吧？」
- 把吉诺拉入房间——当场发生关系——彼此都是第一次
- 剑神反对结婚（认为吉诺无主动性无野心）——吉诺因此觉醒击败加尔后终成夫妻
- 从傲慢的孩子王蜕变为能主动选择人生的成熟剑士

## 能力演变

### 战斗/特殊能力
| 时期 | 能力 |
|------|------|

| 第09卷（18岁） | 剑王级实力——评价为25岁前必成剑帝 |
| 第23卷 | 剑王——剑神流高徒 |
| 蛇足篇 | 剑神流顶级——剑神之妻 |

### 装备与持有物
- 剑（初时一把，第09卷后开始携带两把以防折断）
- 木刀（日常训练）

## 关键人际关系演变

### 艾莉丝·格雷拉特
- 从嫉妒敌对到认同尊重——最大的人物关系变化
- 初遇时被艾莉丝痛打——深感受辱——两年不说话
- 第09卷转变后对艾莉丝视线变得柔和——成为名符其实的对手
- 第20卷与艾莉丝、伊佐露缇三人游玩——关系完全和解

### 吉诺·布里兹（丈夫/表弟）
- 童年时唯一能讨论剑术理论的同龄人，将吉诺当小弟使唤
- 输给艾莉丝后不再把吉诺当小弟——只是刀剑相交互相磨练
- 蛇足篇主动求婚——吉诺因此觉醒击败剑神

### 加尔·法利昂（父亲/师傅）
- 亲生父亲，剑神流师傅
- 对妮娜嫁给吉诺起初反对——认为吉诺无主动性和野心
- 第25卷：父亲加尔之死对妮娜将有重大影响

### 伊佐露缇·克尔埃尔（朋友）
- 曾一同在剑之圣地修行
- 寄信邀请妮娜参加祭典

### 鲁迪乌斯·格雷拉特
- 最初想打倒的对象
- 目睹其实力后震惊——改变了她的价值观

## 重要经历时间线
- **约16岁**：获剑圣称号
- **第09卷·两年前**：初遇艾莉丝——被暴打——尿裤子
- **第09卷（18岁）**：去魔法大学试图打倒鲁迪——目睹巴迪冈迪和鲁迪的压倒性力量——洗心革面
- **第20卷**：前往阿斯拉王国参加爱丽儿加冕祭典——与艾莉丝、伊佐露缇游玩
- **第23卷**：在剑之圣地以守护者姿态出场
- **第24卷**：怀孕——无法支援毕黑利尔
- **第25卷**：父亲加尔之死间接影响
- **蛇足篇**：向吉诺求婚——见证吉诺成为剑神——成为剑神的妻子
