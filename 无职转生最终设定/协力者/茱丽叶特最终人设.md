# 茱丽叶特 / 茱丽 / 朱莉（Juliet / Julie）

## 基本信息
- **全名**：茱丽叶特（Juliette），通称「茱丽（Julie）」；俗称「朱莉」
- **种族**：矿坑族/侏儒族/小人族（Dwarf）
- **性别**：女
- **年龄**：第08卷时六岁 → 第18卷时约十岁 → 第26卷时约成年
- **出身**：父亲圣铁的巴萨尔（铁匠），母亲美丽雪棱的莉莉特拉（装饰品工匠）；因双亲负债沦为奴隶
- **身份**：原奴隶 → 札诺巴的师妹/专属工匠候补 → 人偶职人学徒 → 札诺巴人偶商店干部
- **语言**：兽神语（母语）、人类语（后学习）

## 外貌特征演变
- **第08卷（6岁）**：刚买来时状态极差——营养失调、骨瘦如柴、眼神空虚
- **第10卷（约7岁）**：比以前丰腴，「一年前骨瘦如柴，现在有好好吃东西」
- **第18卷（约10岁+）**：迎来初潮，身体开始发育为成年矮人女性

## 性格与心境演变

### 第08卷——从绝望中获救
- **奴隶市场**：父亲被卖掉，仅剩她一人，极度绝望
- 被鲁迪用兽神语问「你已经想死了吗？」——回答「我不想死」
- 以阿斯拉大铜币一枚被鲁迪买下
- 命名茱丽叶特，本人只记住「茱丽」
- **心理状态**：害怕鲁迪乌斯——将其视为矿坑族童话中「洞穴怪物」的现实版（因为目睹莉妮亚和普露塞娜被绑架监禁后态度大变）
- **餐桌礼仪事件**：用手抓食物、挑食胡萝卜引发鲁迪与札诺巴意见分歧；在自己选择下克服——吃下胡萝卜——露出第一次带有自信和得意的笑容
- 开始称呼鲁迪为「Grand Master」
- 仅一个月即学会以无咏唱方式制造土弹

### 第09卷——学习成长
- 勤勉聪明，像海绵般吸收知识
- 手很灵巧，魔术不行时就动手处理
- 魔力总量压倒性不足，精度也不行
- 想喝巴迪冈迪的酒——矿坑族天性喜好美酒
- 被鲁迪评价为「千载难逢的好买卖」

### 第10-13卷——人偶制作
- 被鲁迪教导读书写字
- 完成了瑞杰路德人偶——水准超越鲁迪的原型
- 学校遇到诺伦，互相道谢
- 诺伦写的瑞杰路德故事听了三遍仍要求再读
- 被诺伦称为「老师」

### 第18卷——独立故事（核心）
- 初潮到来——莉莉雅负责处理教导
- **秘密杰作**：独立制作了一件「超越札诺巴的杰作」——想用展示盒（她称为「床」）包装送给札诺巴
- **误会**：鲁迪以为茱丽去店铺是为了获得自由，实际是要买展示盒
- **真相**：完全没有离开札诺巴的意愿，札诺巴给了她生命的意义和归处
- 札诺巴得知真相后感动落泪
- 鲁迪意识到自己误会了茱丽的心情

### 第19-20卷——与札诺巴的羁绊
- 第19卷：札诺巴被召回西隆王国时留守夏利亚等待
- 第20卷：与札诺巴、金洁三人在夏利亚共同生活——坐在札诺巴大腿上制作人偶
- 毕业典礼时与金洁一同献花——西隆王国习俗

### 第22-24卷——随行
- 第22卷：作为札诺巴助手在魔导铠开发中间接贡献
- 第24卷：随札诺巴往毕黑利尔王国行动，转移魔法阵停摆后赶往斯佩路德村

### 第26卷——成长完成
- 在斯佩路德村：鲁迪跑进村子时喊「大师傅！」
- 十年后担任札诺巴人偶商店干部——多次在会议记录中被提及
- 从札诺巴的仆从成长为人偶商店的重要干部——完成了自身的成长弧

## 能力演变

### 战斗/特殊能力
| 时期 | 能力 |
|------|------|
| 第08卷 | 仅一个月学会无咏唱方式制造土弹（成功率不稳定，魔力快速耗尽） |
| 第09-13卷 | 学习魔术和人偶制作，手极为灵巧；手艺日益精湛 |
| 第18卷 | 制作出「超越札诺巴的杰作」级别的人偶 |
| 第26卷 | 札诺巴人偶商店干部——从制作到管理全面成长 |

## 关键人际关系演变

### 札诺巴·西隆（师傅/主人）
- 被购买后与札诺巴同住——帮忙处理生活杂务
- 很擅长推测札诺巴的需求（被比喻为「金洁那样」）
- 札诺巴是她的师傅、共同生活者——让她从奴隶获得新生的人
- 第18卷核心：制作送给札诺巴的杰作人偶——深深的感恩和依恋
- 第20卷：坐在札诺巴大腿上共同制作人偶
- 第26卷：在札诺巴人偶商店担任干部——延续师徒的传承

### 鲁迪乌斯·格雷拉特（Grand Master）
- 买下她的恩人，教她读书写字、人类语和无咏唱魔术
- 初期害怕（视为「洞穴怪物」），后来建立信赖关系
- 被尊称为「Grand Master」
- 第26卷称呼「大师傅！」

### 金洁·约克
- 与茱丽、札诺巴共同生活于夏利亚
- 三人形成稳固的共同生活关系

### 诺伦·格雷拉特
- 互相称为「老师」——茱丽教诺伦人偶制作，诺伦为茱丽读瑞杰路德故事

## 重要经历时间线
- **出身前**：矿坑族铁匠之女，因双亲负债沦为奴隶，双亲已被卖掉
- **第08卷**：六岁，在瑞姆商会奴隶市场被鲁迪以一枚阿斯拉大铜币买下；命名茱丽叶特；学习无咏唱魔术
- **第09-11卷**：日常学习和人偶制作的早期阶段
- **第13卷**：完成超越鲁迪原型的瑞杰路德人偶；量产开始
- **第18卷**：（核心）初潮到来；独立制作杰作人偶赠予札诺巴；鲁迪的误会与真相揭示
- **第19卷**：札诺巴离开时留守夏利亚等待
- **第20卷**：与札诺巴、金洁三人共同生活；毕业典礼献花
- **第22卷**：在魔导铠开发中作为札诺巴助手间接贡献
- **第24卷**：随札诺巴往毕黑利尔王国行动
- **第26卷**：在斯佩路德村迎接鲁迪；十年后担任札诺巴人偶商店干部
