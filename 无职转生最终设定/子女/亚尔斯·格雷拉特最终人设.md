# 亚尔斯·格雷拉特（Ars Greyrat）

## 基本信息
- **全名**：亚尔斯·格雷拉特（Ars Greyrat）
- **通称**：亚尔斯
- **性别**：男
- **种族**：人族（格雷拉特血统 + 伯雷亚斯血统）
- **家族**：格雷拉特家长男，鲁迪乌斯·格雷拉特与艾莉丝·格雷拉特的长子
- **特征**：红发（继承母亲艾莉丝），体格健壮
- **生日**：甲龙历428年（推定）
- **所属**：格雷拉特家 → 阿斯拉王国（毕业后赴任） → 魔法都市夏利亚

## 外貌特征演变
- **第19卷（新生儿）**：红发——艾莉丝血统的证明
- **第20卷（满周岁）**：尚在婴儿期，被母亲艾莉丝哺乳，被姑姑爱夏溺爱
- **第23卷（约4岁）**：被七星抱住后满脸笑容，将脸埋进七星胸口；鲁迪抱他会嚎啕大哭
- **第24-25卷（童年）**：仅被提及，洛琪希挂念其安危
- **第26卷（13岁）**：成长中，红发少年
- **蛇足篇（10-16岁）**：从持剑幼儿成长为剑神流圣级剑士，体格结实

## 性格与心境演变

### 婴幼儿期（第19-23卷）
- 被爱夏极度溺爱，「被爱夏教育剥夺了思考能力——把只要听爱夏的话就能一切顺利的想法植入了脑海」
- 幼时曾觉得爱夏烦人，但爱夏总是正确的，违逆她几乎都会失败

### 恋心萌芽（蛇足篇·约10-11岁）
- 十岁生日，艾莉丝赠剑「用来守护重要的人」，脑海中立刻浮现爱夏的脸
- 被鲁迪问及想与谁缔结婚约时，自然浮现爱夏的面容
- 不到一年内向爱夏告白
- **关键转折**：家族会议上爱夏被艾莉丝揍，艾莉丝喊「明明教了你要守护的！」——亚尔斯突然理解爱夏也会有走投无路的时候

### 私奔与觉悟期（蛇足篇·约11岁）
- 下定决心：“至今以后要守护爱夏”
- 遵从三个教诲私奔：艾莉丝的「保护重要的人」、爱夏的「有想做的事就该行动」、童话的「相爱的人不该分开」
- 私奔期间剑不离身，将爱夏护在正后方，抵抗魔物和盗贼，在实战中磨练
- **关键转折**：察觉某项变化后不安消失，取而代之觉悟：「就算同归于尽也要守护爱夏」

### 与艾莉丝死斗（蛇足篇·约11-12岁）
- 持剑挡在家门前，眼中含杀意，向父亲母亲们宣言：「喜欢爱夏姐——会一生为守护爱夏而活」
- 被艾莉丝打倒在地仍狙击其足部使其出血；肩膀被砍出深伤、剑连同手腕被斩断仍站立不认输
- 用艾莉丝的剑拖向家门前跪下：「死也不让你们通过」
- 被艾莉丝评价：「不愧是鲁迪乌斯的儿子呢——但那样不能算是尽力保护！」
- **关键转折**：洛琪希告知鲁迪曾向奥尔斯帝德下跪磕头保护家人——亚尔斯崩溃落泪，承认无力，昏迷在血泊中

### 成长与复学（蛇足篇·12-16岁）
- 与爱夏分离后温顺接受，向莉莉雅「请求将女儿托付给他」
- 主动照顾儿子鲁洛伊——学习换尿布、育儿知识
- 与艾莉丝的修行激增，各方面全力以赴
- 四年后获艾莉丝传授光之太刀，剑神流圣级，水魔术火魔术上级
- **毕业时回答**：「已经反省了——但还是跟那天一样地喜欢爱夏——会好好努力的」
- 赴阿斯拉王国迎娶爱夏，在魔法都市夏利亚建立三人家庭

### 晚年（第26卷·老年鲁迪临终）
- 74岁鲁迪临终时，亚尔斯的孙女（曾孙女）菲莉斯——红发——与艾莉絲一模一样；是摘下鲁迪护腕的孩子

## 能力演变

### 战斗/特殊能力
| 时期 | 能力 |
|------|------|
| 婴幼儿（第19-25卷） | 尚未展露战斗能力 |
| 少年初期（蛇足篇·约10岁） | 开始持剑 |
| 私奔期（蛇足篇·约11岁） | 实战杀敌，剑不离身，在实战中磨练技能 |
| 死斗后修行期（蛇足篇·12-16岁） | 剑神流圣级，获传光之太刀，水魔术火魔术上级 |

### 装备与持有物
- 艾莉丝所赠之剑（十岁生日礼物）——「用来守护重要的人」
- 光之太刀（艾莉丝传授）

## 关键人际关系演变

### 爱夏·格雷拉特（姑姑 → 恋人 → 妻子）
- **婴幼儿期**：爱夏极度溺爱，是亚尔斯最亲近的人
- **第20卷**：爱夏「亚尔斯好可爱喔」如口头禅，曾试图让亚尔斯吸自己胸部
- **蛇足篇·恋心期**：十岁时意识到对爱夏的特殊感情，从「听爱夏话」的被支配者变为「要守护爱夏」的守护者
- **私奔期**：遵从爱夏教诲「有想做的事就该行动」
- **分离期（约4年）**：温顺接受分离，努力学习，为将来做准备
- **毕业后**：正式迎娶爱夏，建立家庭

### 艾莉丝·格雷拉特（母亲）
- 哺乳期照顾，赠剑教导「保护重要的人」
- 私奔时的死斗——母子激烈碰撞，艾莉丝砍断亚尔斯的手腕
- **关键转折**：艾莉絲看到亚尔斯的觉悟后认可其成长
- 四年修行期中师徒关系强化，传授光之太刀

### 鲁迪乌斯·格雷拉特（父亲）
- 幼时被父亲抱起会嚎啕大哭
- 蛇足篇私奔事件中激烈对立
- 最终被父亲接纳，毕业后独立建立家庭

### 洛琪希·格雷拉特（二妈）
- 关键时刻告知鲁迪曾向奥尔斯帝德下跪保护家人的事
- 这一信息是亚尔斯觉悟转变的关键触发点

### 七星静香
- 第23卷拥抱亚尔斯，温馨互动

## 重要经历时间线
- **甲龙历428年**：出生于西隆王国期间，鲁迪出外任务
- **第19卷**：诞生，红发男婴——鲁迪返家时已有男婴在家等待
- **第20卷**：满周岁，极度受爱夏溺爱，父亲前往米里斯出差
- **第23卷**：在家，被七星抱住，与父母洗澡
- **第24-25卷**：童年期，在家生活（仅提及）
- **第26卷**：13岁，成长中
- **蛇足篇·约10岁**：十岁生日收到艾莉丝赠剑，不到一年内向爱夏告白
- **蛇足篇·约11岁**：家族会议上爱夏被逼入绝境，亚尔斯与爱夏私奔
- **蛇足篇·约12岁**：被寻获，与艾莉丝死斗，崩溃后归家，复学，与爱夏分离；儿子鲁洛伊出生，亚尔斯成为父亲
- **蛇足篇·约16岁**：毕业，获传光之太刀，剑神流圣级，赴阿斯拉王国迎娶爱夏
- **第26卷·74岁鲁迪临终**：亚尔斯的孙女菲莉斯陪伴在鲁迪身边
