# 齐格哈尔德·萨拉丁·格雷拉特（Sieg）

## 基本信息
- **全名**：齐格弗里德·萨拉丁·格雷拉特（Siegfried Saladin Greyrat），通称「齐格」
- **别名**：齐格哈尔德·格雷拉特（第23卷）、齐格·格雷拉特（第24-26卷）、齐格哈鲁特·萨拉丁·格雷拉特（六面世界3篇）
- **性别**：男
- **种族**：人族（格雷拉特血统 + 拉普拉斯因子）
- **家族**：格雷拉特家第四子（次男），鲁迪乌斯·格雷拉特与希露菲叶特·格雷拉特的儿子
- **特征**：绿色头发（继承拉普拉斯因子），天生腕力强大
- **所属**：北神流 → 北王 → 北帝 → 月光骑士 → 死神骑士（帕库斯建国后）
- **称号**：北帝齐格哈鲁特 → 死神（七大列强之一）

## 外貌特征演变
- **第23卷（新生儿）**：绿发——可能源自拉普拉斯因子，也可能单纯遗传；普通可爱的婴儿，爱哭爱睡，大小便正常
- **第24-25卷（童年）**：仅被提及，在家成长
- **第26卷（11岁）**：成长中，已有独立行动能力
- **六面世界3篇（青年）**：绿发剑士，戴黑色月光骑士头盔，后在帕库斯身边成为「死神骑士」

## 性格与心境演变

### 婴儿期（第23卷）
- 鲁迪评价「比不怎么哭的菈菈或鲁迪一抱就哭的亚尔斯普通许多」
- 在肉体层面受拉普拉斯因子的强烈影响——对任何疾病与环境有抗性
- 奥尔斯帝德保证「这孩子不是拉普拉斯」

### 童年（第24-26卷）
- 在家中被呵护成长

### 青年期——无职的哲学（六面世界3篇）
- 自称「高尚的无职」——认为不工作不是可耻的事
- 从小憧憬「正义的伙伴」芝士超人，听了父亲的英雄故事长大
- 从阿斯拉王立学校毕业后无职一年——什么都没达成
- 白天在城中闲逛，晚上化身「月光骑士」惩治恶党
- 不使用剑，空手战斗——「正义的伙伴不需要武器」当借口
- 情报来自酒馆「醉鬼哥布林」的情报屋乔治——实际是父亲出钱提供
- **核心矛盾**：厌倦自己所为却不敢面对真正想做的事——「我想去找帕库斯」
- 一直没去的原因是担心背叛父亲的期待
- **关键转折**：师傅亚历山大扮演「雷电」阻止其暴走——指出「把攒着的忧郁靠殴打弱小对手来发泄——就像过去的我」
- **与父亲和解**：父亲戴黑面具自称「月影」，喝酒谈心，告诉齐格「爸爸想守护的既不是和平也不是人脉关系——而是你们」
- 父亲唱月光骑士主题歌——前所未有的亲近

### 救出帕库斯（六面世界3篇）
- 抵达王龙王国飞地——从天而降，站在满月前的树顶
- 报上「北帝齐格哈鲁特」——宣誓作为帕库斯的守护者
- 以压倒性力量击败乌巴巴族最强战士海尔佩尔
- 施展「北神流奥义·月光两断」
- 胜利后不杀对手——要求乌巴巴族「成为帕库斯的朋友」
- 对帕库斯说出迟到一年的回答：「我要和你一起走——为了实现你的野心」

### 未来——死神骑士
- 在帕库斯建立的国家中获得「死神」称号
- 继承了七大世界强者中「死神」的位置
- 在三大帝国对峙局势中与帕库斯计划独立

## 能力演变

### 战斗/特殊能力
| 时期 | 能力 |
|------|------|
| 婴儿期 | 拉普拉斯因子赋予的超级抵抗力（免疫疾病与恶劣环境） |
| 少年期 | 天生腕力强大，北神流修行 |
| 青年期 | 北神流北王（世界不到50人），后获北帝称号 |
| 六面世界3篇 | 北神流奥义·月光两断，空手格斗能力 |
| 未来 | 七大列强「死神」——死神骑士 |

### 装备与持有物
- 月光骑士黑色头盔（父亲所赠，作为理解的象征）
- 爱剑（师傅亚历山大代为保管后归还）
- 北帝称号

## 关键人际关系演变

### 鲁迪乌斯·格雷拉特（父亲）
- 童年听父亲讲英雄故事长大
- 毕业无职后一直避开父亲
- **关键转折**：父亲以「月影」身份出现，用笨拙而真情的方式与儿子谈心
- 「你觉得没有意义的时候——反而里面藏着答案」——父亲的理解让齐格重新出发

### 亚历山大·雷白克（师傅）
- 从小教其北神流剑术，认可为北王
- 以「雷电」身份引导其走向正道
- 指出其本质问题：「以正义为借口欺负弱小——就像以前的我」
- 道别时：「你会成为英雄的」
- 齐格拒绝继承师傅意志：「师傅还能活好久——还请师傅继续努力」

### 帕库斯·西隆·Jr（挚友）
- 在阿斯拉王立学校相识，两人在彼此被孤立的灰色学校生活中相依
- 毕业典礼上帕库斯邀请齐格一起建国，齐格当时没有回答
- 时隔一年，齐格赶来的救援——从天而降，压倒性力量守护挚友
- 最终决定一起走：「为了实现你的野心」

### 希露菲叶特·格雷拉特（生母）
- 告别时哭了，齐格是白妈妈亲生的儿子

### 艾莉丝·格雷拉特（红妈妈）
- 告别时：「下定决心了吧——加油」

### 洛琪希·格雷拉特（蓝妈妈）
- 告别时：「既然是你决定的道路就走下去」

### 菈菈·格雷拉特（二姐）
- 齐格道别时罕见地笑了：「今天的齐格很聪明」

### 莉莉·格雷拉特（大妹）
- 对齐格愤怒——本来想让哥哥当助手

## 重要经历时间线
- **第23卷**：出生于魔法都市夏利亚，随父母赴天大陆亚尔切山丘祠堂接受洗礼；奥尔斯帝德保证不是拉普拉斯
- **第24-25卷**：在家成长（仅提及）
- **第26卷**：11岁，全名「齐格弗里德·萨拉丁·格雷拉特」首次揭示
- **六面世界3篇·青年期**：阿斯拉王立学校毕业，无职一年当「月光骑士」
- **六面世界3篇·月光骑士期**：被师傅扮演的「雷电」引导，与父亲和解
- **六面世界3篇·出发**：向三位母亲和姐妹告别，前往找帕库斯
- **六面世界3篇·救出**：从天而降击败乌巴巴族，救出帕库斯
- **未来**：帕库斯建国，齐格成为「死神骑士」，位列七大列强
